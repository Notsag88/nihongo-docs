## Propósito académico del programa

Este repositorio constituye un **programa autodidacta de japonés con estándares académicos profesionales**, diseñado para un estudio riguroso, progresivo y medible.

Su objetivo no es la exposición superficial al idioma, sino la **construcción sólida de competencias lingüísticas reales**, alineadas con el marco del **Japanese-Language Proficiency Test (JLPT)**.

:::info Alcance del programa
El recorrido completo cubre desde la reconstrucción de bases iniciales hasta el dominio funcional de nivel intermedio.
:::

---

## Objetivo formativo (N5 → N3)

El programa está estructurado para permitir al estudiante:

- **Reconstruir las bases (A1)** de forma ordenada, sin lagunas conceptuales.
- Completar **JLPT N5 y N4** con dominio funcional real.
- Alcanzar **JLPT N3** con comprensión efectiva en:
  - lectura
  - escucha
  - gramática aplicada
- Dejar preparado el **puente conceptual y metodológico** hacia **N2 / N1**.

Este recorrido prioriza la **comprensión y el uso**, no la memorización aislada.

---

## Principios docentes del curso

:::info Núcleo metodológico
Los siguientes principios rigen todo el diseño del programa y deben respetarse durante el estudio.
:::

1. **El japonés se aprende en contexto**  
   El foco está en frases, diálogos y textos reales, no en listas descontextualizadas.

2. **Memoria a largo plazo = SRS + uso activo**  
   La repetición espaciada (SRS) se combina siempre con producción y reconocimiento en contexto.

3. **Audición diaria, aunque sea breve**  
   La exposición auditiva constante es indispensable para desarrollar intuición lingüística.

4. **Medición continua del progreso**  
   El avance se valida mediante:
   - checkpoints regulares
   - simulacros
   - bitácora de errores personal

## Material base integrado

:::note Material de apoyo
Este programa integra material externo como soporte, siempre subordinado al recorrido académico propuesto.
:::

- **Koi – Manual Básico de Japonés (Satori, Yoko Nakazawa)**  
  Texto central para el reinicio A1 y el desarrollo de JLPT N5.

- **Japanese Kanji Flash Cards Vol. 1 (Tuttle)**  
  Apoyo inicial para kanji, **siempre trabajado con frases**, nunca de forma aislada.

El uso de estos materiales es **guiado**, no libre.

---

## Cómo utilizar este repositorio

Para obtener el máximo beneficio del programa, se recomienda:

- Seguir **estrictamente el orden de la barra lateral**.
- Mantener una **bitácora de errores**, actualizada semanalmente.
- Cerrar cada semana de estudio con un **checkpoint breve**, aunque el progreso haya sido lento.
- Priorizar la **regularidad** sobre la cantidad diaria de contenido.

:::warning Advertencia importante
Avanzar sin cerrar checkpoints o sin registrar errores suele generar lagunas que reaparecen en niveles superiores.
:::
