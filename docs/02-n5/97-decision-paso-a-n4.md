# Decisión de avance: N5 → N4

:::info Propósito académico
Este documento establece si el estudiante:
- puede avanzar a N4 sin riesgo estructural
- o debe reforzar áreas específicas antes de continuar
:::

---

# Criterio institucional de avance

El paso a N4 **NO depende solo de aprobar un examen**.

Se requiere dominio funcional en:

- gramática base
- vocabulario activo
- lectura sin bloqueo
- comprensión auditiva básica
- producción escrita simple pero correcta

---

# Requisitos mínimos (obligatorios)

Marca **solo si se cumple sin esfuerzo excesivo**:

- [ ] El verbo siempre va al final sin pensar
- [ ] Distingo claramente です vs verbo
- [ ] Uso は / が / を / に / で correctamente
- [ ] Conjugo pasado afirmativo y negativo sin dudar
- [ ] Uso あります / います sin traducir “tener”
- [ ] Leo kanji N5 sin bloqueo visual
- [ ] Puedo escribir 10–12 frases nuevas sin copiar

:::warning Regla institucional
Si fallan **2 o más puntos**, NO se recomienda avanzar a N4.
:::

---

# Resultado de la decisión

Seleccionar **una sola opción**:

- ✅ **N5 cerrado – avanzar a N4**
- ⚠️ **Refuerzo parcial antes de N4**
- ❌ **Repetir módulos críticos de N5**

---

# Si el resultado es “Refuerzo parcial”

Áreas típicas a reforzar:
- partículas (especialmente に / で)
- pasado negativo
- あります / います
- lectura real con kanji

:::note Nota docente
Reforzar NO es retroceder.
Es asegurar que N4 no se vuelva frustrante.
:::

---

# Conclusión

Avanzar a N4 con base débil:
❌ genera abandono  
❌ memorismo  
❌ confusión estructural  

Avanzar con base sólida:
✔️ acelera N4  
✔️ mejora lectura real  
✔️ reduce errores futuros
