# Diagnóstico estructural de N5

:::info Propósito docente
Este documento NO evalúa “nota”.
Evalúa **patrones de error**.
:::

---

# Áreas evaluadas

Analizar cada área y marcar **solo si el error aparece con frecuencia**.

---

## 1. Partículas

Errores típicos detectables:
- confundir に / で
- usar を con existencia
- omitir は

Resultado:
- [ ] Dominado
- [ ] Inestable
- [ ] Débil

---

## 2. Tiempo verbal

Errores frecuentes:
- mezclar presente y pasado
- olvidar ました
- usar です con verbos

Resultado:
- [ ] Dominado
- [ ] Inestable
- [ ] Débil

---

## 3. あります / います

Errores críticos:
- usar あります con personas
- expresar posesión con を
- usar は en vez de が

Resultado:
- [ ] Dominado
- [ ] Inestable
- [ ] Débil

:::warning Alerta docente
Esta área mal aprendida afecta N4 y N3 directamente.
:::

---

## 4. Adjetivos

Errores frecuentes:
- あついでした
- しずかい
- きれいなです

Resultado:
- [ ] Dominado
- [ ] Inestable
- [ ] Débil

---

## 5. Kanji

Evaluar honestamente:
- bloqueo visual
- lectura lenta
- dependencia de furigana

Resultado:
- [ ] Dominado
- [ ] Inestable
- [ ] Débil

---

## 6. Producción escrita

Problemas comunes:
- copiar modelos
- evitar escribir
- frases muy cortas o repetitivas

Resultado:
- [ ] Dominado
- [ ] Inestable
- [ ] Débil

---

# Síntesis final

:::info Interpretación
- 0–1 áreas débiles → N5 sólido
- 2–3 áreas débiles → refuerzo dirigido
- 4+ áreas débiles → repetir N5 parcialmente
:::

---

# Acción obligatoria

Para cada área marcada como “Débil”:

1. Registrar errores reales en la bitácora  
2. Crear tarjetas Anki específicas  
3. Releer el módulo correspondiente  
4. Volver a producir frases nuevas

Esto **convierte errores en progreso real**.
