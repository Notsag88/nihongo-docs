# Qué significa “N5 cerrado” en este curso

N5 cerrado NO significa:
❌ aprobar un simulacro  
❌ memorizar reglas  
❌ reconocer vocabulario aislado  

Significa:
✔️ construir frases correctas  
✔️ leer sin bloqueo  
✔️ comprender estructuras básicas  
✔️ tener base sólida para N4  

---

# Competencias alcanzadas

Al cerrar N5, el estudiante puede:

- construir oraciones simples correctas
- usar partículas básicas funcionalmente
- expresar tiempo (presente / pasado)
- describir con adjetivos
- expresar existencia y posesión
- leer textos cortos con kanji N5
- comprender audio lento y claro

# Próxima etapa: N4

En N4 se incorporan:
- gramática menos explícita
- lectura más densa
- escucha real más rápida
- menos ayudas visuales

Por eso, este cierre es **crítico**.
