# Rol de la cultura en el aprendizaje del japonés

:::info Núcleo del módulo
La cultura en japonés **no es un complemento**.
Es parte activa de cómo se habla, se escucha y se interpreta el idioma.
:::

En niveles iniciales (N5), la cultura permite:

- entender **por qué se dice algo así**
- evitar traducciones mentales incorrectas
- interpretar silencios, omisiones y elecciones gramaticales
- comprender el uso real de expresiones básicas

Sin esta base cultural:
- la gramática se vuelve mecánica
- el listening se vuelve confuso
- la lectura pierde intención

---

# Qué cubre Cultura Premium N5

:::info Alcance del nivel
Este módulo cubre **solo lo esencial**, pero de forma rigurosa.
:::

En N5 trabajamos:

- forma japonesa de organizar la información
- noción de contexto y omisión
- cortesía básica (sin formalismo avanzado)
- comportamiento lingüístico cotidiano
- relación entre cultura y gramática básica

No trabajamos todavía:
- keigo avanzado
- normas empresariales
- pragmática de alto nivel

Eso pertenece a **N4–N3**.

---

# Principio cultural central (clave desde N5)

:::info Principio japonés fundamental
El japonés prioriza:
**contexto + armonía + claridad implícita**
por encima de la explicitud verbal.
:::

Esto explica:

- por qué se omite el sujeto
- por qué は no significa “yo”
- por qué muchas frases parecen “incompletas” al traducir
- por qué repetir información es poco natural

---

# Relación directa con la gramática N5

:::note Conexión crítica
Muchos “errores gramaticales” de N5
son en realidad **errores culturales**.
:::

Ejemplos:

- uso excesivo de わたしは
- necesidad constante de aclarar el sujeto
- incomodidad con frases cortas
- traducción literal de estructuras españolas

Este módulo corrige eso **desde la raíz**.

---

# Cómo usar el módulo de Cultura

:::info Metodología recomendada
:::

Cada archivo cultural debe trabajarse así:

1. Leer completo (sin analizar)
2. Identificar la idea central
3. Relacionarla con:
   - una estructura gramatical vista
   - un error propio frecuente
4. Escribir **1 nota personal** (máx. 3 líneas)

:::warning Advertencia metodológica
No memorices cultura.
Reflexionala y observá cómo aparece en el idioma.
:::

- concreto
- directamente aplicable al idioma

# Relación con JLPT (y con la vida real)

:::note Importante
El JLPT no evalúa cultura explícitamente,
pero **la asume** en cada ejercicio.
:::

Quien domina cultura básica:
- entiende mejor listening
- comete menos errores gramaticales
- progresa más rápido en N4

---

# Criterio de avance del módulo

Antes de cerrar Cultura N5 deberías poder:

- [ ] aceptar frases sin sujeto explícito
- [ ] no forzar traducciones literales
- [ ] entender por qué se omite información
- [ ] relacionar cultura con gramática básica

Si esto no está claro:
→ revisar antes de pasar a N4.

# Cierre del index

:::info Cierre docente
La cultura japonesa **no se agrega después**.
Se integra desde el inicio.
Este módulo asegura que tu japonés crezca
con base real, no artificial.
