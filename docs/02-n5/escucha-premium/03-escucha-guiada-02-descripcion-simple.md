# Objetivo de esta escucha guiada

:::info Núcleo del ejercicio
Esta escucha entrena la comprensión de **descripciones simples** y de estructuras con  
**あります / います**, fundamentales para N5 y absolutamente críticas para N4.
:::

Al finalizar este archivo deberías poder:

- identificar si se describe un **lugar**, una **persona** o un **objeto**
- reconocer あります / います en tiempo real
- distinguir información principal de detalles
- seguir una descripción corta sin traducir palabra por palabra

---

# Tipo de audio recomendado

:::info Nivel adecuado
Elegí audios con estas características:
:::

- 2–5 oraciones
- velocidad clara (natural, no robótica)
- presencia explícita de:
  - lugares
  - objetos
  - personas o animales
- uso de あります / います

Ejemplos de temática:
- una habitación
- una casa
- un parque
- una escuela
- un lugar con personas u objetos

:::note Nota docente
Este tipo de audio es **estructural**, no narrativo.
No hay historia: hay **descripción**.
:::

---

# Escucha modelo (estructura esperada)

:::info Modelo de referencia
Ejemplo del tipo de contenido que vas a escuchar.
No memorizar.
:::

> へやに つくえが あります。  
> つくえの うえに ほんが あります。  
> へやに ねこが います。

Idea global suficiente:
- descripción espacial
- existencia
- uso correcto de が / に

---

# Protocolo de trabajo (obligatorio)

:::warning Regla metodológica
No avances al siguiente paso si el anterior no está cumplido.
:::

---

## Paso 1 — Escucha global (sin texto)

- Escuchá el audio completo una vez
- No pauses
- No intentes traducir

Respondé mentalmente:
- ¿se describe un lugar?
- ¿hay personas / animales?
- ¿hay objetos?

No importa el detalle todavía.

---

## Paso 2 — Escucha funcional

Escuchá nuevamente y tratá de identificar:

- si aparece **あります** o **います**
- al menos **un objeto**
- al menos **un lugar**

:::note Nota docente
No identificar todo es normal.
Identificar **una sola estructura** ya es progreso.
:::

---

## Paso 3 — Confirmación con texto

:::info Uso correcto del texto
El texto se usa para **verificar hipótesis**, no para entender desde cero.
:::

- Leé el texto
- Confirmá:
  - qué existe
  - dónde existe
- Marcá:
  - palabras desconocidas
  - partículas mal comprendidas

Solo ahora:
✔️ diccionario  
✔️ bitácora  

---

## Paso 4 — Lectura en voz alta + escucha

- Leé el texto en voz alta una vez
- Volvé a escuchar el audio
- Leé nuevamente acompañando el ritmo

Objetivo:
- sincronizar oído y estructura
- reforzar verbo final

---

# Errores frecuentes en esta escucha

:::warning Errores típicos
:::

- ❌ confundir あります y います
- ❌ perderse intentando traducir todo
- ❌ no identificar el lugar
- ❌ ignorar las partículas に / が

:::note Nota docente
Si no entendés quién existe o dónde,
la frase no se comprendió correctamente.
:::

---

# Duración recomendada

:::info Carga óptima
:::

- ⏱ 15 minutos
- 1 solo audio
- repetir el mismo audio 2 días seguidos

No acumules audios distintos.

---

# Señales claras de progreso

:::info Indicadores positivos
:::

- reconocés あります / います apenas aparecen
- anticipás que viene un objeto o persona
- las descripciones ya no suenan caóticas
- podés explicar el contenido con palabras simples

---

# Mini-checkpoint

:::note Verificación antes de avanzar
:::

- [ ] Distingo cosas vs personas al escuchar
- [ ] Reconozco あります / います en audio
- [ ] Identifico el lugar descrito
- [ ] No necesito traducir todo para entender

Si fallan 2 o más:
→ repetir con otro audio descriptivo similar.

---

# Nota final del profesor

:::info Cierre pedagógico
La descripción con existencia es uno de los **patrones más frecuentes del japonés real**.

Si esto se automatiza en N5,
N4 deja de ser intimidante.
:::
