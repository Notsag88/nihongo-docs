# Objetivo de esta escucha integradora

:::info Núcleo del ejercicio
Esta escucha consolida **rutina**, **descripción** y **existencia** en un solo flujo.
Es el cierre natural de Escucha Premium N5.
:::

Al finalizar este archivo deberías poder:

- seguir una narración corta con **rutina diaria**
- identificar **adjetivos** en contexto
- reconocer **あります / います** dentro de una historia simple
- mantener coherencia global sin traducir frase por frase

---

# Tipo de audio recomendado

:::info Nivel adecuado
Elegí audios con estas características:
:::

- 4–7 oraciones
- una sola persona como foco
- combinación de:
  - rutina (まいにち / あさ / よる)
  - descripción (adjetivos い / な)
  - existencia (あります / います)

Temas típicos:
- descripción de una casa
- un día normal + cómo es el lugar
- alguien hablando de su entorno cotidiano

:::note Nota docente
Este tipo de audio **mezcla funciones**.
La clave no es entender todo, sino **no perder la estructura**.
:::

---

# Escucha modelo (estructura esperada)

:::info Modelo de referencia
Ejemplo del tipo de contenido que vas a escuchar.
No memorizar.
:::

> まいにち、7時に おきます。  
> いえは あたらしいです。  
> へやに テレビが あります。  
> よる、テレビを 見ます。

Ideas globales:
- rutina diaria clara
- descripción estable
- existencia integrada naturalmente

---

# Protocolo de trabajo (obligatorio)

:::warning Regla metodológica
Si intentás traducir todo,
vas a perder la idea central.
:::

---

## Paso 1 — Escucha global (sin texto)

- Escuchá una vez completa
- No pauses
- No tomes notas

Respondé mentalmente:
- ¿es una rutina?
- ¿describe un lugar o una persona?
- ¿hay cosas que “existen”?

No busques vocabulario todavía.

---

## Paso 2 — Identificación de bloques

Escuchá otra vez y detectá **bloques funcionales**:

- tiempo (まいにち / あさ / よる)
- descripción (adjetivos)
- existencia (あります / います)
- acción final (verbo)

:::note Nota docente
Si reconocés los bloques,
aunque no cada palabra,
la escucha está bien hecha.
:::

---

## Paso 3 — Confirmación con texto

:::info Uso correcto del texto
El texto sirve para **confirmar estructura**, no para aprender de memoria.
:::

- Leé el texto completo
- Marcá:
  - adjetivos
  - あります / います
  - verbos finales
- Verificá:
  - verbo siempre al final
  - adjetivo antes del sustantivo

---

## Paso 4 — Lectura rítmica

- Leé el texto en voz alta
- Volvé a escuchar el audio
- Leé nuevamente siguiendo el ritmo

Objetivo:
- naturalizar la mezcla de funciones
- evitar “resetear” la frase al oír algo nuevo

---

# Errores frecuentes en esta escucha

:::warning Errores típicos
:::

- ❌ perderse cuando aparece un adjetivo
- ❌ confundir descripción con acción
- ❌ bloquearse al oír あります / います
- ❌ intentar traducir todo

:::note Nota docente
El japonés real **mezcla funciones constantemente**.
Este ejercicio te prepara para eso.
:::

---

# Duración recomendada

:::info Carga óptima
:::

- ⏱ 20 minutos
- 1 solo audio
- repetir el mismo audio 2–3 días

Si necesitás más tiempo, **es normal** en este punto.

---

# Señales claras de progreso

:::info Indicadores positivos
:::

- seguís la idea general sin esfuerzo
- reconocés descripción sin “pensar gramática”
- あります / います ya no sorprende
- el audio fluye como un bloque

---

# Mini-checkpoint final (Escucha N5)

:::note Verificación de cierre
:::

- [ ] Entiendo rutinas simples al escuchar
- [ ] Reconozco adjetivos en contexto
- [ ] Identifico existencia (あります / います)
- [ ] Mantengo la idea global sin traducir
