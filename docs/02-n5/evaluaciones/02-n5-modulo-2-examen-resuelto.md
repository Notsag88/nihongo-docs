# Propósito del resuelto

:::info Enfoque docente
Este resuelto valida el dominio del **pasado** y la **rutina diaria**.
Un error aquí suele indicar:
- confusión temporal
- mala selección de partículas
- traducción literal desde español
:::

---

# Parte A — Gramática (pasado) — Resuelto

### Ejercicio

1. きのう、パンを ___。  
2. コーヒーを ___。  

*(pistas: comer / no beber)*

---

### Respuestas correctas

1. きのう、パンを **たべました**。  
2. コーヒーを **のみませんでした**。

---

### Comentario docente

- Pasado afirmativo: **ます → ました**
- Pasado negativo: **ません → ませんでした**

:::warning Error típico
❌ たべますでした  
✔️ たべました
:::

Si aparece este error:
→ repasar **regularidad verbal** (no hay excepciones aquí).

---

# Parte B — Producción escrita — Criterios

### Ejemplo de respuesta correcta

- まいにち、7じに おきます。  
- きのう、いえで テレビを みました。  
- あさ ごはんを たべませんでした。  

---

### Qué se evalúa (rubrica real)

✔️ coherencia temporal  
✔️ expresiones de tiempo al inicio  
✔️ verbo final consistente  
✔️ partículas funcionales  

❌ mezclar presente y pasado  
❌ omitir tiempo cuando es relevante  

:::note Indicador positivo
Si el estudiante **piensa primero “cuándo”**
y luego conjuga, va bien.
:::

---

# Parte C — Kanji (lectura y significado) — Resuelto

### Ítems

- 昨日  
- 7時  
- 帰ります  

---

### Respuestas esperadas

- **昨日（きのう）**: ayer  
- **7時（しちじ／ななじ）**: las 7 en punto  
- **帰ります（かえります）**: volver (forma ます)

---

### Comentario docente

- No se evalúa escritura perfecta
- Sí se evalúa **reconocimiento inmediato**

:::warning Alerta
Si se reconoce el kanji pero no la palabra,
falta **integración léxica**.
:::

---

# Parte D — Listening (criterio)

### Evaluación esperada

El estudiante debe poder identificar:
- **cuándo** ocurre la acción
- **qué** acción ocurre
- **dónde** ocurre

No se exige transcripción exacta.

:::info Regla docente
Comprender la situación > comprender cada palabra
:::

---

# Evaluación global del Módulo 2

## Aprobado si:

- ≥ 75% correcto
- pasado usado sin dudas
- rutina expresada con claridad

## No aprobado si:

- mezcla tiempos sin control
- evita el pasado por inseguridad
- traduce literalmente

---

# Acciones correctivas obligatorias

1. Registrar errores temporales en la bitácora
2. Crear tarjetas Anki:
   - presente ↔ pasado
   - afirmativo ↔ negativo
3. Reescribir 5 frases en pasado
4. Releer:
   - pasado de verbos
   - expresiones de tiempo
