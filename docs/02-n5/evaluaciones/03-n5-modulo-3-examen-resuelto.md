# Corrección docente – Visión general

:::info Evaluación resuelta
Esta corrección explica:
- la respuesta correcta
- el porqué estructural
- el error típico asociado
:::

El objetivo no es solo “ver si está bien”, sino **entender por qué**.

---

# Parte A — Identificación de adjetivos

1. あつい → **い**  
2. しずか → **な**  
3. きれい → **な**  
4. おおきい → **い**  
5. にぎやか → **な**

:::note Nota docente
Los adjetivos な **no terminan en い**, aunque algunos lo parezcan en significado.
:::

---

# Parte B — Forma correcta (predicado)

6. この へやは **あついです**。  
7. あの まちは **しずかです**。  
8. この みせは **やすいです**。  
9. きのうは **さむかったです**。

:::warning Error típico
❌ あついでした  
✔️ あつかったです  

Los adjetivos い **conjugan**, no usan でした directamente.
:::

---

# Parte C — Uso antes del sustantivo

10. **しずかな へや**  
11. **おおきい いえ**  
12. **にぎやかな まち**

:::info Regla clave
- い → va directo
- な → **な obligatorio**
:::

---

# Parte D — Pasado de adjetivos

13. あつい → **あつかったです**  
14. しずか → **しずかでした**  
15. やすい → **やすかったです**

:::note Patrón estructural
- い → cambia la terminación
- な → usa でした
:::

---

# Parte E — Producción escrita (ejemplos válidos)

:::info Nivel esperado
A continuación se muestran **ejemplos posibles**.
No son las únicas respuestas correctas.
:::

Ejemplos de frases correctas:

- この へやは あついです。  
- あの まちは しずかでした。  
- おおきい いえに すんでいます。  
- きれいな みせで かいものします。  
- きのうは さむかったです。  
- にぎやかな まちが すきです。  
- この ほんは いいです。  
- あたらしい いえは きれいです。

✔️ Uso correcto de い / な  
✔️ Pasado correcto  
✔️ Estructura japonesa natural  

---

# Parte F — Kanji (reconocimiento)

16. **大きい** → grande  
17. **小さい** → pequeño  
18. **新しい** → nuevo  
19. **古い** → viejo  

:::note Criterio N5
Estos kanji son de **reconocimiento prioritario**.
La escritura perfecta no es obligatoria.
:::

# Recomendaciones docentes

1. Crear tarjetas Anki de contraste:
   - あつい → あつかった
   - しずか → しずかな
2. Reescribir 5 frases del examen sin mirar el modelo
3. Prestar atención a:
   - ❌ い + でした
   - ❌ な omitido

Este módulo es **clave para N4**.
