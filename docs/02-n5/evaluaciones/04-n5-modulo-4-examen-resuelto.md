# Evaluación N5 · Módulo 4 – Corrección docente  
**Tema:** あります / います – existencia, ubicación y posesión

:::info Criterio de corrección
No se evalúa traducción literal.
Se evalúa:
- distinción animado / inanimado
- estructura japonesa correcta
- uso funcional de partículas
:::

---

# Parte A — Selección correcta

1. へやに ほんが **あります**。  
   ✔️ libro = objeto → あります

2. こうえんに ねこが **います**。  
   ✔️ gato = ser vivo → います

3. がっこうに せんせいが **います**。  
   ✔️ persona → います

4. つくえの うえに かばんが **あります**。  
   ✔️ objeto → あります

5. いえに ひとが **います**。  
   ✔️ persona → います

:::note Regla aplicada
Si se mueve por voluntad propia → います  
Si no → あります
:::

---

# Parte B — Forma negativa

6. へやに テレビが **ありません**。  
7. こうえんに こどもが **いません**。  
8. わたしに くるまが **ありません**。

✔️ Uso correcto de:
- が para lo que existe
- に para el lugar / poseedor

❌ Error frecuente evitado:
- わたしは くるまを ありません

---

# Parte C — Pasado (existencia)

9. きのう、へやに ねこが **いました**。  
10. さっき、がっこうに せんせいが **いました**。

✔️ Forma pasada correcta:
- あります → ありました
- います → いました

---

# Parte D — Ubicación (orden correcto)

11.  
**つくえの したに ほんが あります。**  
(Hay un libro debajo del escritorio)

12.  
**こうえんに こどもが います。**  
(Hay niños en el parque)

:::info Recordatorio estructural
El orden correcto no es español.
La partícula define la función.
:::

---

# Parte E — Posesión

13. Tengo un libro.  
→ **わたしに ほんが あります。**

14. Mi amigo tiene un perro.  
→ **ともだちに いぬが います。**

15. Yo no tengo auto.  
→ **わたしに くるまが ありません。**

✔️ Posesión = existencia en alguien  
❌ Nunca usar を para “tener”

---

# Parte F — Producción escrita (modelo correcto)

Ejemplos válidos (uno posible conjunto):

- へやに テレビが あります。  
- こうえんに ねこが います。  
- きのう、いえに ともだちが いました。  
- わたしに ほんが あります。  
- いえの なかに いすが あります。  
- さっき、がっこうに せんせいが いました。  

✔️ Cumple:
- ubicación
- posesión
- pasado
- partículas correctas

---

# Parte G — Kanji (reconocimiento)

16. 上 — arriba  
17. 中 — dentro  
18. 外 — afuera  
19. 前 — delante  

✔️ Reconocimiento suficiente para N5  
✖️ No es obligatorio escribirlos de memoria
