---
title: Errores típicos con adjetivos (N5)
---

:::info Núcleo del nivel
Esta sección consolida los **errores más frecuentes y persistentes** en el uso de adjetivos japoneses en nivel N5.  
Su objetivo no es solo corregir, sino **prevenir la fossilización** de errores estructurales.
:::

## Principio docente clave

:::tip Regla pedagógica central
En japonés, los adjetivos **no se comportan como adjetivos españoles**.  
Muchos errores provienen de tratarlos como “sustantivo + ser + adjetivo”.
:::

Comprender **por qué** el error ocurre es más importante que memorizar la corrección.

---

## Tabla de errores típicos (análisis causal)

| Error incorrecto | Causa real | Forma correcta | Explicación docente |
|-----------------|-----------|----------------|---------------------|
| あついでした | Tratar el adjetivo い como si necesitara です para conjugar | あつかったです | Los adjetivos い **se conjugan solos** |
| しずかい | Confundir adjetivo な con adjetivo い | しずか | しずか es な, **no se le agrega い** |
| きれいなです | Doble marcación (な + です) | きれいです | な **solo** se usa antes del sustantivo |
| おおきいな いえ | Mezcla de tipos (い + な) | おおきい いえ | Los adjetivos い **no usan な** |

---

## Análisis por tipo de error

### 1. Error de conjugación (adjetivos い)

:::warning Error estructural frecuente
❌ あついでした  
:::

✔️ あつかったです  

Explicación:
- La い **se elimina**
- Se reemplaza por かった
- です es **opcional** y solo marca formalidad

---

### 2. Error de clasificación (い vs な)

:::warning Error de tipo
❌ しずかい  
:::

✔️ しずか  

Explicación:
- しずか **no es un adjetivo い**
- Nunca termina en い
- Usa です para conjugar tiempo

---

### 3. Error de doble marcación (な + です)

:::warning Error por sobrecorrección
❌ きれいなです  
:::

✔️ きれいです  

Explicación:
- な **solo conecta con el sustantivo**
- No se usa cuando el adjetivo es predicado

---

### 4. Error por mezcla de sistemas

:::warning Error mixto
❌ おおきいな いえ  
:::

✔️ おおきい いえ  

Explicación:
- おおきい ya funciona solo
- Agregar な indica **confusión conceptual**

---

## Regla premium de prevención

:::tip Regla de oro (memorizar)
Si el adjetivo:
- **cambia de forma** → es い  
- **usa です para cambiar tiempo** → es な
:::

Esta regla explica **todos los casos N5**.

---

## Estrategia correctiva obligatoria (SRS)

:::note Protocolo docente
Si un error aparece **más de 2–3 veces**, es señal de riesgo de fossilización.
:::

Acción obligatoria:
- Crear tarjeta Anki **con contraste**:

**Frente:**  
❌ あついでした  

**Reverso:**  
✔️ あつかったです  
→ Los adjetivos い se conjugan solos

---

## Criterio de dominio

Antes de avanzar, deberías poder:

- [ ] Identificar si un adjetivo es い o な sin mirar tablas
- [ ] Conjugar adjetivos い en pasado correctamente
- [ ] Usar な solo delante del sustantivo
- [ ] Detectar estos errores al leer frases ajenas

Si fallan 2 o más → revisar:
- Tablas canónicas de adjetivos
- Contraste い vs な
