---
title: Estructura de frases de ubicación
---

:::info Núcleo del nivel
Las frases de ubicación son una **aplicación directa** del sistema de existencia japonés  
(あります / います) y del uso funcional de partículas.

Este modelo es **obligatorio** para JLPT N5.
:::

## 1. Estructura canónica de ubicación

La estructura básica para indicar que algo o alguien **existe en un lugar** es:

> **[Lugar] に [Objeto / Persona] が あります／います**

Esta estructura responde a la pregunta implícita:  
> “¿Qué existe, y dónde?”

---

## 2. Ejemplos claros (modelo estándar)

:::info Ejemplos canónicos
:::

- つくえの うえに ほんが あります。  
  (Hay un libro sobre el escritorio)

- へやに ねこが います。  
  (Hay un gato en la habitación)

Observá que:
- **に** marca el lugar de existencia  
- **が** introduce el elemento que existe  
- el verbo final depende de si el referente es animado o no

---

## 3. Función real de las partículas

:::note Partículas en contexto
Las partículas **no se eligen por traducción**, sino por función.
:::

| Partícula | Función real |
|----------|--------------|
| **に** | lugar donde algo existe |
| **が** | elemento que aparece / existe |

En este tipo de frases:
- **に** es prácticamente obligatoria  
- **が** es la norma para introducir nueva información

---

## 4. Relación con あります / います

:::info Conexión estructural
La estructura de ubicación **no puede separarse** de la distinción:
:::

- **あります** → objetos, cosas, ideas  
- **います** → personas y animales  

Ejemplos contrastivos:
- へやに テレビが **あります**。  
- へやに ねこが **います**。

Un error en el verbo implica **un error conceptual**, no solo gramatical.

---

## 5. Errores frecuentes (alerta docente)

:::warning Errores sistemáticos del español
❌ usar を como si fuera objeto directo  
❌ usar は sin razón discursiva  
❌ traducir “estar” literalmente
:::

Ejemplos incorrectos:
- ❌ へやに ほんを あります  
- ❌ へやは ねこが います *(sin contraste previo)*

Forma correcta (N5):
- ✔️ へやに ほんが あります  
- ✔️ へやに ねこが います  

---

## 6. Ejercicio guiado (recomendado)

:::tip Práctica mínima diaria
Escribe 6–8 frases usando:
- diferentes lugares
- al menos 2 objetos
- al menos 2 seres animados
:::

Modelo:
- こうえんに こどもが います。  
- いえに テレビが あります。

---

## 7. Criterio de dominio

:::note Checklist de avance
Antes de continuar, deberías poder:
- [ ] Usar に como lugar de existencia
- [ ] Usar が para introducir lo que existe
- [ ] Elegir correctamente あります / います
- [ ] Detectar errores de ubicación en ejemplos incorrectos
:::

Si alguno falla → revisar este bloque antes de avanzar.
