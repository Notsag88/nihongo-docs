---
title: Expresar posesión con あります / います
---

:::info Núcleo del nivel
En japonés **no existe un verbo equivalente a “tener”**.

La posesión se expresa como un caso particular de **existencia**.
Este concepto es obligatorio para JLPT N5 y se reutiliza en N4 y N3.
:::

## 1. Idea clave (criterio japonés)

En japonés:

> **“Tener” = “Existe algo en alguien”**

No se describe una acción, sino un **estado de existencia asociado a una persona**.

---

## 2. Estructura canónica de posesión

La estructura básica es:

> **[Persona] に [Objeto / Persona] が あります／います**

Donde:
- **に** marca el “lugar conceptual” (la persona)
- **が** introduce lo que existe
- el verbo depende de si el referente es animado o no

---

## 3. Ejemplos básicos (modelo N5)

:::info Ejemplos canónicos
:::

- わたしに くるまが あります。  
  (Tengo un auto)

- ともだちに こどもが います。  
  (Mi amigo tiene hijos)

Observá que:
- no aparece ningún verbo equivalente a “tener”
- la persona se trata como **marco de existencia**, no como sujeto activo

---

## 4. Comparación conceptual con el español

:::note Cambio mental necesario
:::

**Español**  
> Yo tengo un libro.

**Japonés (estructura real)**  
> En mí, existe un libro.

Este cambio de perspectiva es **fundamental** para evitar errores futuros.

---

## 5. Elección correcta del verbo

:::info Regla aplicada
:::

- **あります** → objetos, cosas, ideas  
- **います** → personas, animales  

Ejemplos:
- わたしに ほんが **あります**。  
- かぞくに いぬが **います**。

---

## 6. Errores frecuentes (alerta docente)

:::warning Errores típicos del principiante
❌ わたしは くるまを あります  
❌ わたしが ほんを もっています *(mezcla prematura)*  
❌ traducir “tener” palabra por palabra
:::

Forma correcta (N5):
- ✔️ わたしに くるまが あります  
- ✔️ わたしに ほんが あります  

---

## 7. Relación con otros contenidos N5

:::note Conexión curricular
Este uso se conecta directamente con:
- estructura de ubicación
- uso de に como marco
- introducción de información nueva con が
- distinción あります / います
:::

Si uno de estos puntos falla, la frase de posesión **se rompe**.

---

## 8. Ejercicio guiado (recomendado)

:::tip Práctica mínima
Escribe 6–8 frases expresando posesión:
- 4 con objetos
- 2 con personas o animales
:::

Modelo:
- わたしに スマホが あります。  
- ともだちに ねこが います。

---

## 9. Criterio de dominio

:::note Checklist de avance
Antes de continuar, deberías poder:
- [ ] Expresar posesión sin usar “tener”
- [ ] Usar に para la persona correctamente
- [ ] Usar が para lo poseído
- [ ] Elegir あります / います sin dudar
:::

Si alguno falla → revisar este bloque antes de avanzar.
