# Tabla Canónica de Kanji N5 – Core

:::info Criterio académico
Esta tabla contiene los **kanji fundamentales del nivel N5** que todo estudiante debe
**reconocer con seguridad** al finalizar el nivel.

- No todos deben escribirse de memoria  
- Todos deben leerse sin bloqueo  
- Cada kanji debe aparecer en palabras y frases reales
:::

---

## Regla de dominio (obligatoria)

:::warning Condición de aprendizaje real
Cada kanji de esta tabla debe cumplir:

- [ ] Al menos **3 frases propias**
- [ ] **1 tarjeta Anki**
- [ ] Aparición en **1 lectura real**

Si no cumple estas condiciones, **no se considera aprendido**.
:::
