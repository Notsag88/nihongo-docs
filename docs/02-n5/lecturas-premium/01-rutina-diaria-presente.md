# Objetivo de la lectura

:::info Objetivo académico
Esta lectura tiene como finalidad que puedas:

- Leer un texto continuo sin bloqueo
- Reconocer estructuras N5 en contexto real
- Integrar tiempo + lugar + acción
- Reforzar presente y pasado
- Ganar confianza lectora
:::

:::note Nivel esperado
No se espera comprensión total.  
Se espera **comprensión global y estructural**.
:::

---

# Texto principal

:::info Texto
わたしは まいにち 6時に おきます。  
あさ ごはんを たべて、コーヒーを のみます。  

7時に いえを でて、がっこうに いきます。  
がっこうで べんきょうします。  

ひるに ともだちと ごはんを たべます。  
ごご、ほんを よみます。  

よる、いえに かえります。  
テレビを みて、11時に ねます。  
:::

---

# Comprensión global (sin traducir palabra por palabra)

:::tip Paso 1 – Idea general
Respondé mentalmente (o por escrito):

- ¿De quién habla el texto?
- ¿Qué tipo de texto es?
- ¿Ocurre en presente o pasado?
:::

---

# Guía de comprensión dirigida

:::info Preguntas clave
Respondé en español o japonés simple:

1. ¿A qué hora se levanta la persona?
2. ¿A dónde va a las 7?
3. ¿Qué hace al mediodía?
4. ¿Qué hace por la noche?
:::

---

# Foco gramatical

:::info Gramática en uso
Este texto refuerza:

- forma ます
- expresiones de tiempo
- uso de に / で
- secuencia de acciones con て
:::

Ejemplos clave:
- 6時に おきます  
- がっこうで べんきょうします  
- テレビを みて、ねます
