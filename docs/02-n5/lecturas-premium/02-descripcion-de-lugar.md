# Propósito académico de esta lectura

:::info Núcleo del nivel
Esta lectura entrena la **descripción estática** en japonés, una habilidad central para:
- comprensión lectora N5
- uso correcto de あります / います
- integración de adjetivos con sustantivos
:::

A diferencia de una narración, aquí **no hay acciones principales**, sino **existencia, ubicación y características**.

---

# Texto principal

:::info Texto base (N5 realista)
<div class="example">

わたしの いえは ちいさいです。  
いえの なかに へやが ふたつ あります。  

ひとつは ねる へやです。  
その へやは しずかで、きれいです。  

もう ひとつの へやは いま べんきょうする へやです。  
つくえの うえに ほんが あります。  
へやの なかに テレビは ありません。  

いえの そとに ちいさい にわが あります。  
にわに ねこが います。

</div>
:::

---

# Comprensión global (sin traducir palabra por palabra)

:::note Lectura estratégica
Antes de analizar detalles, asegurate de poder responder **mentalmente**:
:::

- ¿Cómo es la casa?
- ¿Cuántas habitaciones hay?
- ¿Qué hay dentro de la casa?
- ¿Qué no hay?
- ¿Hay personas o animales?

Si necesitás traducir cada palabra para responder → **todavía no es lectura funcional**.

---

# Análisis guiado (enfoque docente)

## 1. Uso de あります / います

:::info Punto clave
Este texto refuerza la **distinción conceptual**, no solo gramatical.
:::

- へやが あります → cosa inanimada  
- にわが あります → cosa inanimada  
- ねこが います → ser vivo

:::warning Error típico
❌ にわに ねこが あります  
✔️ にわに ねこが います
:::

---

## 2. Adjetivos en contexto real

Observá el uso **natural**, no forzado:

- ちいさい いえ  
- しずかで、きれいです  

:::note Nota docente
En japonés real:
- los adjetivos **no siempre aparecen aislados**
- se encadenan naturalmente con で
:::

---

## 3. Uso de negación con existencia

Frase clave del texto:

> へやの なかに テレビは ありません。

Esto muestra:
- contraste afirmativo / negativo
- uso de **は** para negar algo esperado

:::info Regla de lectura
En frases negativas de existencia:
- が introduce lo que existe
- は suele marcar lo que **no** existe
:::

# Ejercicio de consolidación (obligatorio)

## A) Producción escrita

Escribí **6–8 frases nuevas** describiendo:
- tu casa
- una habitación
- un lugar conocido

Requisitos:
- al menos 3 frases con あります
- al menos 1 frase con います
- al menos 2 adjetivos

---

## B) Relectura activa

1. Lee el texto completo en voz alta
2. Marca las frases con あります / います
3. Lee una vez más sin detenerte
