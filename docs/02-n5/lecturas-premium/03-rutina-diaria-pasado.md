# Propósito académico de esta lectura

:::info Núcleo del nivel
Esta lectura entrena simultáneamente:
- pasado en forma ました / ませんでした
- expresiones temporales
- coherencia temporal en un texto continuo
- lectura de rutina real (no frases sueltas)
:::

Es uno de los **puntos donde más estudiantes se rompen** si no está bien trabajado.

---

# Texto principal

:::info Texto base (N5 realista, pasado)
<div class="example">

きのうは へいじつでした。  
わたしは あさ 6じに おきました。  

7じに あさごはんを たべて、  
8じに いえを でました。  

9じから 5じまで、がっこうで べんきょうしました。  
ひるは ともだちと レストランで たべました。  

よる、いえに かえって、  
テレビを すこし みました。  

でも、コーヒーは のみませんでした。  
よる 11じに ねました。

</div>
:::

---

# Comprensión global (nivel lector)

:::note Lectura estratégica
Antes de analizar detalles, respondé mentalmente:
:::

- ¿Cuándo ocurre el texto?
- ¿Qué hace la persona por la mañana?
- ¿Dónde pasa la mayor parte del día?
- ¿Qué acción NO hace?
- ¿A qué hora termina el día?

Si necesitás traducir cada verbo → todavía no hay automatización del pasado.

---

# Análisis guiado (enfoque docente)

## 1. Uso consistente del pasado

:::info Punto clave
Todo el texto mantiene **coherencia temporal absoluta**.
:::

Observá:
- おきました  
- たべました  
- でました  
- べんきょうしました  
- のみませんでした  
- ねました  

:::warning Error típico
❌ mezclar ます y ました en el mismo texto  
✔️ todo el bloque narrativo en pasado
:::

---

## 2. Expresiones de tiempo (posición correcta)

Ejemplos del texto:

- きのうは  
- あさ 6じに  
- 7じに  
- 9じから 5じまで  
- よる 11じに  

:::info Regla de lectura
En japonés:
- el tiempo suele aparecer **al inicio**
- no interrumpe la estructura verbal
:::

---

## 3. Encadenamiento de acciones (forma て – reconocimiento)

Frase clave:

> 7じに あさごはんを たべて、  
> 8じに いえを でました。

:::note Nota docente
Aquí aparece la **forma て** solo como:
- conector temporal
- sin explicación gramatical aún

En N5:
- se reconoce
- no se analiza en profundidad todavía
:::

---

## 4. Uso intencional de は en contraste

Frase clave:

> コーヒーは のみませんでした。

Esto indica:
- contraste con acciones anteriores
- algo esperado que **no ocurrió**

:::info Regla premium
は aparece frecuentemente en:
- contraste
- negación contextual
:::

# Vocabulario clave del texto

:::info Vocabulario activo
No memorizar listas. Integrar en frases.
:::

| Japonés | Significado |
|------|-------------|
| へいじつ | día de semana |
| あさ | mañana |
| ひる | mediodía |
| よる | noche |
| でる | salir |
| かえる | volver |
| ねる | dormir |

# Ejercicio de consolidación (obligatorio)

## A) Producción escrita (pasado)

Escribí **8–10 frases** sobre **tu día de ayer**, usando:

- al menos 5 verbos en pasado
- al menos 3 expresiones de tiempo
- 1 acción negativa

Ejemplo:
- きのう、よる テレビを みませんでした。

---

## B) Lectura activa en voz alta

1. Lee el texto completo sin pausar
2. Subrayá mentalmente los verbos en pasado
3. Lee una vez más manteniendo ritmo estable

# Relación con el JLPT N5

Este tipo de texto aparece en:
- comprensión lectora
- listening narrativo simple
- preguntas de rutina diaria
:::

Fallar aquí suele indicar:
- pasado mal automatizado
- problemas de coherencia temporal

# Criterio de dominio

Antes de avanzar, deberías poder:

- [ ] Seguir la secuencia temporal sin traducir
- [ ] Reconocer pasado afirmativo y negativo
- [ ] Entender contrastes con は
- [ ] Leer el texto completo con fluidez

Si fallan 2 o más:
→ repetir lectura + escritura guiada.
