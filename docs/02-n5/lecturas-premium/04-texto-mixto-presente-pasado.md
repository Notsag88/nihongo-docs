# Propósito académico de esta lectura

:::info Núcleo del nivel
Esta lectura entrena la **descripción estable de personas**, integrando:
- adjetivos い / な en contexto real
- estructuras con です
- contraste con は
- lectura descriptiva sin acción principal
:::

Es una habilidad clave para:
- presentaciones personales
- comprensión lectora básica
- listening descriptivo del JLPT N5

---

# Texto principal

:::info Texto base (N5 descriptivo)
<div class="example">

この ひとは たなかさんです。  
たなかさんは 30さいです。  

たなかさんは せが たかくて、  
かみが みじかいです。  

いつも しずかで、  
とても まじめな ひとです。  

がっこうで せんせいです。  
でも、いえでは しずかでは ありません。  

たなかさんは コーヒーが すきですが、  
おさけは のみません。

</div>
:::

---

# Comprensión global (sin traducción literal)

:::note Lectura estratégica
Antes de analizar detalles, respondé mentalmente:
:::

- ¿Quién es la persona?
- ¿Cómo es físicamente?
- ¿Cómo es su personalidad?
- ¿Dónde trabaja?
- ¿Qué contraste aparece?

Si necesitás traducir palabra por palabra → la lectura aún no es funcional.

---

# Análisis guiado (enfoque docente)

## 1. Uso de adjetivos en contexto real

:::info Punto clave
Los adjetivos aparecen **integrados**, no aislados.
:::

Ejemplos del texto:
- せが たかい  
- かみが みじかい  
- しずか  
- まじめな ひと  

:::warning Error típico
❌ pensar que los adjetivos “van sueltos”  
✔️ siempre forman parte de una estructura nominal
:::

---

## 2. Encadenamiento descriptivo con て

Frase clave:

> せが たかくて、かみが みじかいです。

:::note Nota docente
Aquí la forma て:
- conecta características
- **no expresa tiempo**
- es descriptiva

En N5:
- se reconoce
- no se analiza aún como gramática formal
:::

---

## 3. Contraste con は (uso discursivo)

Frase clave:

> がっこうで せんせいです。  
> でも、いえでは しずかでは ありません。

:::info Regla premium
は se usa para:
- contrastar contextos
- oponer información esperada vs real
:::

---

## 4. Expresar gustos (reconocimiento)

Frase:

> コーヒーが すきですが、  
> おさけは のみません。

Esto introduce:
- estructura de gusto
- contraste suave con が / は

:::note Nota docente
Aquí:
- が marca preferencia
- は marca exclusión
:::

# Vocabulario clave del texto

:::info Vocabulario activo
Integrar siempre en frases propias.
:::

| Japonés | Significado |
|------|-------------|
| せ | altura |
| かみ | cabello |
| みじかい | corto |
| たかい | alto |
| まじめ | serio |
| すき | gustar |

# Ejercicio de consolidación (obligatorio)

## A) Producción escrita

Escribí **6–8 frases** describiendo:
- una persona real o imaginaria

Requisitos:
- mínimo 3 adjetivos
- al menos 1 contraste con は
- 1 frase de gusto o preferencia

Ejemplo:
- この ひとは しずかですが、いえでは にぎやかです。

---

## B) Lectura activa

1. Lee el texto completo en voz alta
2. Identificá mentalmente:
   - adjetivos
   - contrastes
3. Relee sin detenerte

---

# Relación con el JLPT N5

:::info Evaluación real
Este tipo de texto aparece en:
- comprensión lectora descriptiva
- listening de presentación personal
- ejercicios de “qué tipo de persona es”
:::

Errores aquí suelen indicar:
- adjetivos mal comprendidos
- uso incorrecto de は

# Criterio de dominio

Antes de avanzar, deberías poder:

- [ ] Describir una persona sin copiar modelos
- [ ] Reconocer adjetivos en lectura
- [ ] Entender contrastes con は
- [ ] Leer el texto completo con fluidez

Si fallan 2 o más:
→ repetir lectura + producción escrita.
