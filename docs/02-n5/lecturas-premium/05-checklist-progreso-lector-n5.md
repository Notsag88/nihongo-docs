# Propósito académico de esta lectura

:::info Núcleo del nivel
Esta lectura entrena la **comprensión informativa**, distinta de:
- narración (rutina)
- descripción (personas / lugares)

Aquí el foco está en:
- entender información objetiva
- seguir una explicación breve
- reconocer estructura expositiva básica
:::

Este tipo de texto aparece frecuentemente en:
- materiales educativos
- avisos simples
- textos introductorios reales

---

# Texto principal

:::info Texto base (N5 informativo)
<div class="example">

にほんの がっこうは、  
4がつに はじまります。  

がっこうは、  
しゅうに 5かい あります。  

がくせいは、  
あさ 9じから ごご 3じまで  
べんきょうします。  

ひるは、  
がっこうで ごはんを たべます。  

にちようびは、  
がっこうは ありません。

</div>
:::

# Comprensión global (lectura funcional)

:::note Lectura estratégica
Sin traducir palabra por palabra, respondé:
:::

- ¿Cuándo empieza la escuela?
- ¿Cuántos días por semana hay clases?
- ¿En qué horario estudian?
- ¿Qué pasa el domingo?

Si necesitás volver al texto para cada respuesta → normal.  
Si necesitás traducir todo → falta lectura expositiva.

# Análisis guiado (enfoque docente)

## 1. Estructura informativa clara

:::info Punto clave
Este texto sigue un patrón típico japonés:
:::

- tema general → にほんの がっこう
- información concreta en frases cortas
- ausencia de opiniones personales

Esto facilita:
- lectura rápida
- extracción de datos

---

## 2. Uso de あります en sentido institucional

Frases clave:
- がっこうは しゅうに 5かい あります。
- にちようびは、がっこうは ありません。

:::info Regla premium
あります no solo expresa existencia física, también:
- eventos
- frecuencia
- actividades programadas
:::

---

## 3. Expresiones de tiempo (lectura natural)

Observá:
- 4がつに
- あさ 9じから
- ごご 3じまで
- にちようびは

:::note Nota docente
En textos informativos:
- el tiempo organiza la información
- no siempre aparece al inicio absoluto
:::

---

## 4. Uso funcional de は

Ejemplos:
- がっこうは
- がくせいは
- にちようびは

:::info Regla de lectura
は introduce:
- tema general
- contraste implícito (domingo vs días de clase)
:::

# Ejercicio de consolidación (obligatorio)

## A) Producción informativa

Escribí **5–7 frases** informativas sobre:
- tu escuela
- tu trabajo
- o tu rutina semanal

Requisitos:
- estilo neutral (sin opinión)
- al menos 2 expresiones de tiempo
- al menos 1 uso de あります

Ejemplo:
- がっこうは しゅうに 3かい あります。

---

## B) Relectura estratégica

1. Lee el texto completo una vez
2. Identificá mentalmente:
   - datos numéricos
   - expresiones de tiempo
3. Relee sin pausar

# Criterio de dominio

Antes de avanzar, deberías poder:

- [ ] Extraer información sin traducir todo
- [ ] Reconocer uso informativo de あります
- [ ] Seguir la estructura del texto
- [ ] Leer con ritmo estable

Si fallan 2 o más:
→ repetir lectura + escritura informativa.
