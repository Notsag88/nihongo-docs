---
title: Checklist de dominio – Lectura N5
---

## Propósito académico

Este checklist define **criterios objetivos** para determinar si la **competencia lectora N5** está realmente adquirida.

> :::info Núcleo del nivel
> Avanzar de nivel sin dominar la lectura básica genera
> dificultades acumulativas en N4 y N3.
> :::

Este documento no es motivacional.  
Es **diagnóstico y decisorio**.

---

## 1. Comprensión general (sin traducción)

Marca solo si puedes hacerlo **de forma automática**:

- [ ] Entiendo el sentido general de un texto N5 corto sin traducir palabra por palabra
- [ ] Identifico la acción principal de una frase rápidamente
- [ ] Reconozco si el texto es narrativo, descriptivo o funcional (aviso)

---

## 2. Identificación estructural

- [ ] Localizo el verbo principal sin dificultad
- [ ] Distingo tiempo (presente / pasado)
- [ ] Reconozco partículas clave (は / が / を / に / で)
- [ ] Entiendo la función de あります / います en contexto

---

## 3. Lectura funcional real

- [ ] Comprendo avisos simples (prohibiciones, horarios, instrucciones)
- [ ] Entiendo textos cortos sobre rutina diaria
- [ ] Puedo explicar con mis palabras qué se espera del lector

---

## 4. Fluidez mínima esperada

- [ ] Leo frases N5 sin detenerme en cada palabra
- [ ] No necesito diccionario en la primera lectura
- [ ] Uso el diccionario solo para confirmar palabras clave

---

## 5. Producción vinculada a lectura

- [ ] Puedo reformular una lectura en 2–3 frases propias
- [ ] Puedo crear frases nuevas usando estructuras vistas en lectura
- [ ] Registro errores de lectura en la bitácora

---

## 6. Decisión académica

### Puedes avanzar a N4 si:

- Todas las secciones tienen **máximo 1 casilla sin marcar**
- Comprendes lecturas N5 sin ansiedad ni bloqueo
- El proceso de lectura es **controlado y consciente**

### No debes avanzar si:

- Traduces mentalmente todo
- Te bloqueas ante kanji conocidos
- No reconoces estructuras básicas

> :::warning Decisión docente
> Avanzar sin cumplir este checklist **no ahorra tiempo**:
> lo traslada como dificultad futura.
> :::

---

## Acción posterior obligatoria

Antes de iniciar **N4**:

1. Revisar textos fallados
2. Crear tarjetas Anki de:
   - estructuras
   - errores de interpretación
3. Releer estrategias de lectura N5

---

## Resultado esperado

Si cumples este checklist:

✔️ La lectura N5 queda **formalmente cerrada**  
✔️ Estás preparado para lecturas extensas N4  
✔️ El japonés empieza a “fluir” sin traducción constante
