# Qué es “leer japonés” en nivel inicial (visión académica)

:::warning Qué NO es leer japonés
En un enfoque serio, **leer japonés NO significa**:

❌ traducir palabra por palabra  
❌ entender cada kanji de memoria  
❌ depender de romaji o furigana constantemente  
❌ buscar equivalencias exactas con el español  
:::

:::info Qué SÍ significa leer japonés
Leer japonés **sí significa**:

✔️ identificar la idea principal del texto  
✔️ reconocer estructuras ya estudiadas  
✔️ tolerar ambigüedad controlada  
✔️ integrar gramática + vocabulario + kanji en contexto  
✔️ avanzar aunque no entiendas el 100%  
:::

Este bloque de lecturas es donde el idioma **empieza a vivir**.

# Objetivo del bloque Lecturas Premium N5

:::info Objetivo académico
Al finalizar las lecturas N5 deberías poder:

- Leer textos cortos **sin bloqueo**
- Reconocer kanji N5 automáticamente
- Comprender la **estructura lógica** de frases reales
- Inferir significado por contexto
- Preparar una transición sólida a N4
:::

:::note Nota docente
Si solo estudiaste gramática y vocabulario, **todavía no sabés japonés**.  
La lectura es lo que convierte conocimiento en idioma.
:::

# Principios metodológicos (no negociables)

## 1. Lectura antes que traducción

:::tip Principio clave
Primero se entiende:
- ¿de qué trata el texto?
- ¿qué pasa?
- ¿quién hace qué?

La traducción (si se hace) es **posterior y parcial**.
:::

---

## 2. Comprensión progresiva (no total)

:::info Nivel esperado N5
En N5:
- 60–80% de comprensión **es excelente**
- buscar el 100% genera frustración y dependencia
:::

---

## 3. Kanji en contexto real

:::info Uso real del kanji
El kanji:
- no se memoriza aquí
- se **reconoce**
- se refuerza naturalmente por repetición contextual
:::

---

## 4. Relectura estratégica

:::tip Método recomendado
Un texto se trabaja así:
1. lectura global
2. lectura focalizada
3. relectura fluida

No se “lee una sola vez”.
:::

# Tipos de lectura incluidos en este bloque

:::info Tipología de textos
Este bloque incluye lecturas cuidadosamente graduadas:
:::

## A) Lecturas narrativas simples
- rutinas diarias
- acciones encadenadas
- tiempo pasado / presente

## B) Lecturas descriptivas
- personas
- lugares
- objetos
- uso intensivo de adjetivos

## C) Lecturas funcionales
- mensajes
- mini diálogos
- descripciones cotidianas reales

Todas usan:
- gramática N5 consolidada
- kanji core N5
- vocabulario de alta frecuencia

# Estructura estándar de cada lectura premium

:::info Formato fijo de los archivos
Cada archivo de lectura premium incluye:

1. Texto principal (sin romaji)  
2. Objetivo de lectura  
3. Guía de comprensión  
   - preguntas conceptuales  
   - no traducción literal  
4. Foco gramatical  
5. Foco de kanji  
6. Errores típicos esperables  
7. Tarea activa  
   - reescritura  
   - resumen  
   - frases propias  
8. Criterio de dominio
:::

---

# Cómo estudiar una lectura (protocolo recomendado)

:::tip Protocolo sugerido
⏱ Tiempo recomendado por lectura: **30–45 minutos**
:::

1. Leer el texto completo (sin detenerse)
2. Marcar lo no entendido (sin resolver aún)
3. Responder preguntas de comprensión
4. Releer el texto
5. Extraer:
   - 2–3 frases modelo
   - 1–2 palabras o kanji dudosos
6. Crear Anki **solo de lo relevante**

# Criterio para avanzar de Lecturas N5

:::warning Checkpoint obligatorio
Antes de pasar a N4 deberías poder:

- [ ] Leer textos cortos sin romaji
- [ ] Reconocer kanji N5 sin bloqueo
- [ ] Entender la idea general sin traducir todo
- [ ] Releer con mayor fluidez que la primera vez
- [ ] Escribir un resumen simple en japonés
:::

Si esto no se cumple → **no avanzar**.
