title: N5 · Módulo 4 · Vocabulario de ubicación y existencia
---

# Propósito del módulo

Este vocabulario sirve para **activar correctamente** las estructuras:

- あります / います
- ubicación con に
- existencia con が
- posesión indirecta

No se aprende como lista.  
Se aprende **en frases completas y funcionales**.

---

# 1. Lugares frecuentes (N5)

Estos sustantivos aparecen constantemente en frases de existencia.

| Japonés | Significado |
|-------|-------------|
| へや | habitación |
| いえ | casa |
| がっこう | escuela |
| こうえん | parque |
| みせ | tienda |

Ejemplos en contexto:

- がっこうに せんせいが います。  
  (Hay profesores en la escuela)

- いえに へやが あります。  
  (Hay habitaciones en la casa)

> Nota docente:  
> Los lugares **van marcados con に**,  
> no con で (que se usa para acciones).

---

# 2. Objetos comunes (cosas inanimadas)

Estos sustantivos se usan siempre con **あります**.

| Japonés | Significado |
|-------|-------------|
| ほん | libro |
| つくえ | escritorio |
| いす | silla |
| テレビ | televisión |
| くるま | auto |

Ejemplos:

- へやに テレビが あります。  
  (Hay una televisión en la habitación)

- いえに くるまが あります。  
  (Hay un auto en casa / Tengo un auto)

> Observación importante:  
> Esta misma estructura sirve para **posesión**.

---

# 3. Personas y animales (seres vivos)

Estas palabras se usan siempre con **います**.

| Japonés | Significado |
|-------|-------------|
| ひと | persona |
| こども | niño |
| ねこ | gato |
| いぬ | perro |

Ejemplos:

- こうえんに こどもが います。  
  (Hay niños en el parque)

- いえに ねこが います。  
  (Hay un gato en casa)

⚠️ Error típico:
> ❌ ねこが あります  
> ✔️ ねこが います  

---

# 4. Integración con posesión (clave N5)

Recuerda esta estructura:

> [Persona] に [Elemento] が あります／います

Ejemplos:

- わたしに くるまが あります。  
  (Tengo un auto)

- わたしに いぬが います。  
  (Tengo un perro)

> Nota docente:  
> No se usa “tener”.  
> Algo **existe en relación a alguien**.

---

# 5. Regla docente de activación (obligatoria)

Cada palabra nueva debe cumplir:

1. **1 frase propia** (no copiada)
2. **1 tarjeta Anki** (palabra + frase)
3. **1 aparición real** (lectura o audio)

Si falta una → **no está aprendida**.

---

# 6. Ejercicio guiado recomendado

Escribe **8–10 frases** usando:

- al menos 3 lugares
- al menos 3 objetos
- al menos 2 personas o animales

Ejemplos modelo:
- へやに ほんが あります。
- こうえんに いぬが います。
- わたしに くるまが あります。

---

# 7. Criterio de dominio del vocabulario

Este módulo está consolidado cuando:

- Usas あります／います sin dudar
- No confundes に y で
- Expresas posesión sin traducir “tener”
- Reconoces estas palabras en lectura básica

Si hay errores → repite antes de avanzar.
