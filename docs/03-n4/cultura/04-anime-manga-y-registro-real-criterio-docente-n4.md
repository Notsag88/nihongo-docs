# Anime, manga y registro real (criterio docente N4)

:::info Núcleo del nivel
En N4, muchos estudiantes ya consumen:
- anime
- manga
- videojuegos
- streams

Eso puede ser una ventaja enorme… o una fuente de errores graves.

El objetivo de este archivo no es “prohibir”.
Es darte criterio profesional para distinguir:
- japonés de ficción
- japonés real
- registro apropiado según contexto
:::

---

## 1. El problema no es el anime: es el registro

:::note
El japonés de ficción suele maximizar:
- personalidad
- emoción
- dramatismo
- rasgos sociales extremos

Eso no es “incorrecto”.
Es **marcado**.

El error es hablar así en contextos neutros.
:::

---

## 2. Tres categorías útiles (criterio operativo)

:::info Clasificación premium
A) Registro neutro usable (alto valor)  
B) Registro marcado (usar con cuidado)  
C) Registro ficticio / no trasladable (reconocer, no usar)
:::

---

## 3. A) Registro neutro usable (alto valor)

Ejemplos típicos:
- 〜です／〜ます (base)
- 〜と思います (opinión)
- 〜てもいいですか (permiso)
- 〜てください (petición)
- すみません／ありがとうございます

:::note
Si tu consumo de anime refuerza estas formas, es positivo.
:::

---

## 4. B) Registro marcado (usar con cuidado)

Ejemplos de señales:
- formas demasiado masculinas o “duras”
- jerga de grupos específicos
- estilo de personaje

:::warning
El problema no es “suena raro”.
El problema es:
- sonar socialmente torpe
- sonar agresivo sin querer
- sonar infantil o teatral
:::

Regla N4:
- reconocerlo
- entenderlo
- no adoptarlo como base

---

## 5. C) Registro ficticio / no trasladable

Incluye:
- muletillas de personaje
- lenguaje arcaico o teatral
- estilos extremos (samurai, princesa, villano)

:::info
Tu meta es:
- comprenderlo en consumo
- pero construir tu japonés sobre registro neutro
:::

---

## 6. El error típico intermedio: copiar “frases cool”

:::warning Error cultural
Copiar frases memorables y usarlas en conversación real.

Resultado frecuente:
- suena exagerado
- el interlocutor se distancia
- te encasillan como inmaduro lingüísticamente
:::

---

## 7. Estrategia premium para usar anime/manga como herramienta

:::info Método docente
1) Elige escenas de conversación cotidiana  
2) Extrae frases neutras y repetibles  
3) Convierte a tu registro (です／ます) si corresponde  
4) Registra vocabulario y patrones en Anki  
:::

Ejemplo de transformación:
- (ficción marcada) → (registro neutro)
No se trata de “corregir”, sino de **adaptar**.

---

:::note Marca solo si es cierto
- [ ] Distingo registro neutro vs marcado
- [ ] No copio muletillas de personaje
- [ ] Uso anime/manga para reforzar estructuras N4 reales
- [ ] Puedo transformar frases marcadas a neutras
- [ ] Entiendo que “natural” depende del contexto, no del show
:::

Si usas cultura pop con criterio, acelera el aprendizaje.
Si la copias sin filtro, sabotea tu japonés.

N4 es el nivel donde ese filtro se vuelve obligatorio.
:::
