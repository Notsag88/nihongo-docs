# Errores culturales del estudiante intermedio (N4)

:::info Núcleo del nivel
En N4 aparecen errores nuevos:

ya no son “errores de forma”.
Son errores de:
- intención
- tono
- adecuación social
- lectura de contexto

Lo peligroso es que:
la frase puede estar gramaticalmente bien,
pero culturalmente mal.
:::

---

## 1. Ser demasiado directo al rechazar

:::warning Error típico
Decir “no puedo” de forma frontal y cerrada.
:::

Estrategia japonesa:
- suavizar
- justificar
- dejar espacio

Ejemplos preferibles N4:
- すみません、ちょっと…  
- 今は忙しいので、また今度…  
- 今日は用事があるので…。

---

## 2. Opinar como sentencia

:::warning Error típico
Hablar como si tu evaluación fuera absoluta:
- “esto es malo”
- “esto es inútil”
:::

Preferibles N4:
- ちょっと難しいと思います。  
- 私には合わないと思います。  
- こちらのほうがいいと思います。

---

## 3. Exigir explicitación total (bloqueo por omisión)

:::warning
En lectura/escucha, frustrarse porque:
“no dice el sujeto”
“no termina la frase”
:::

Solución premium:
- aceptar omisión como norma
- buscar conectores
- inferir intención probable

---

## 4. Usar demasiada cortesía “forzada”

:::note
No es un error ser cortés.
El error es usar fórmulas largas sin control y sonar artificial.
:::

N4 natural:
- frases breves
- tono estable
- amortiguación con と思います／ちょっと

---

## 5. Mezclar registros (neutro + marcado)

:::warning
Un problema común:
usar estructura neutra y meter una forma marcada.
Resultado: inconsistencia, pérdida de naturalidad.
:::

Regla premium:
- construir una base neutra estable
- después, expandir registros en niveles superiores

---

## 6. Confundir “suave” con “ambiguo”

:::info
El japonés puede ser suave y claro a la vez.
Suavidad ≠ falta de intención.

Ejemplo:
- 今は忙しいので、できません。  
Es claro, pero no agresivo.
:::

---

## 7. Entrenamiento premium (bitácora cultural)

:::info Método
Crea una bitácora con 3 columnas:

1) Lo que dije/interpreté (literal)
2) Problema cultural (tono / contexto / registro)
3) Versión japonesa natural (N4)

Repite semanalmente.
:::
