# Examen N4 – Módulo 01  
## Forma て – Uso real y control funcional

:::info Instrucciones
Este examen evalúa **dominio funcional** de la forma て en nivel N4.

No se evalúa:
- memorización de tablas
- repetición mecánica

Se evalúa:
- control automático
- comprensión del sentido
- elección correcta según contexto

Tiempo sugerido: 20–25 minutos  
No usar diccionario.
:::

---

## Parte 1 – Transformación automática (て-forma)

Convierte correctamente los siguientes verbos a **forma て**.

1. 食べる →  
2. 行く →  
3. 書く →  
4. 話す →  
5. 待つ →  

:::note
Aquí se evalúa **automatización**.  
Dudar excesivamente indica que la forma aún no está integrada.
:::

---

## Parte 2 – Uso funcional (permiso y prohibición)

Elige la opción **más natural** según el contexto.

### 1.
ここで写真を＿＿＿もいいですか。  
a) 撮る  
b) 撮って  
c) 撮った  

### 2.
この部屋に＿＿＿はいけません。  
a) 入る  
b) 入って  
c) 入った  

:::warning
No se evalúa solo la forma gramatical, sino si **la estructura cumple la función comunicativa correcta**.
:::

---

## Parte 3 – Conexión de acciones (secuencia lógica)

Lee el siguiente texto y responde.

> 朝ごはんを食べて、シャワーを浴びて、会社に行きます。

Pregunta:  
¿Cuál es el orden correcto de las acciones?

---

## Parte 4 – Comprensión funcional (lectura breve)

Lee con atención.

> 雨が降っているので、家にいます。  
> テレビを見て、少し休みます。

Pregunta:  
¿Qué hace la persona **y por qué**?

---

## Parte 5 – Producción controlada (forma て)

Escribe **dos frases completas**:

1. Una frase usando forma て para **pedir permiso**.  
2. Una frase usando forma て para **prohibición**.

:::warning
Frases incompletas o traducidas literalmente del español **no obtienen puntaje completo**.
:::

---

## Criterio de aprobación del módulo

:::info Referencia
Este examen se considera aprobado si:
- las transformaciones son correctas
- se eligen correctamente permiso y prohibición
- se comprende la secuencia de acciones
- las frases producidas son naturales en N4
:::
