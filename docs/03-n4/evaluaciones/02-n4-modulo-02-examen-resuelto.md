# Resolución comentada – Módulo 02  
## Forma て aplicada: criterio y uso real

:::info Criterio de corrección
En este módulo no basta con “saber la forma”.
Se evalúa **cuándo** y **para qué** se usa.

Cada corrección explica el razonamiento esperado en N4.
:::

---

## Parte 1 – Peticiones

### 1.
ちょっと＿＿＿ください。  
✔️ **b) 待って**

Explicación:
- 〜てください es la estructura fija para peticiones
- Forma diccionario aquí es incorrecta funcionalmente

---

### 2.
ここに名前を＿＿＿ください。  
✔️ **b) 書いて**

:::note Docente
Este uso debe ser **automático** en N4.
Si se duda, la forma て aún no está consolidada.
:::

---

## Parte 2 – Detección de uso incorrecto

### 3.
✔️ **a) ドアを閉めて、テレビを見ます。**

Explicación:
- Describe acciones propias en secuencia
- b) es incorrecta porque mezcla petición con acción propia
- c) es gramatical, pero cambia el matiz (〜てから)

:::warning
En N4 se evalúa **intención**, no solo corrección formal.
:::

---

## Parte 3 – Simultaneidad vs secuencia

Frase:
> 音楽を聞いて、勉強します。

Respuesta correcta:
✔️ **b) Estudia mientras escucha música**

Explicación:
- Esta estructura suele expresar **acción simultánea**
- Si fuera secuencia estricta, se esperaría contexto adicional

---

## Parte 4 – Comprensión funcional

Texto:
> テレビを見て、ごはんを食べて、シャワーを浴びました。

Respuesta correcta:
✔️ **b) Secuencia de acciones**

Explicación:
- Acciones naturalmente ordenadas
- No se realizan al mismo tiempo

:::note
La forma て **no tiene un solo significado fijo**.
El contexto define la relación.
:::

---

## Parte 5 – Producción controlada

### Ejemplos válidos (no únicos)

Petición:
- ここに座ってください。  
- もう一度言ってください。

Acción simultánea:
- 音楽を聞いて、勉強します。  
- コーヒーを飲んで、新聞を読みます。

:::info Criterio
Se evalúa:
- estructura correcta
- relación lógica entre acciones
- naturalidad N4
:::

---

## Evaluación global del Módulo 02

:::info Resultado
✔️ Aprobado si:
- se usan peticiones sin error
- se distingue simultaneidad de secuencia
- se comprende la intención del hablante
- la producción es natural

❌ No aprobado si:
- se confunden usos de la forma て
- se responde solo por “regla memorizada”
- se traduce literalmente del español
:::

---

:::note Cierre docente
Cuando este módulo está sólido,
la forma て deja de ser un problema gramatical
y se convierte en una **herramienta expresiva**.

Desde aquí, el N4 empieza a sonar real.
:::
