# Resolución comentada – Módulo 05  
## Intención con criterio N4

:::info Criterio de corrección
En este módulo se evalúa **decisión consciente**.

El error típico es:
- confundir futuro con intención
- usar つもりです como simple tiempo verbal

La corrección explica el **matiz real**.
:::

---

## Parte 1 – Elección de estructura

### 1.
来年、日本に＿＿＿つもりです。  
✔️ **a) 行く**

Explicación:
- つもりです se construye con forma diccionario
- Indica plan o decisión previa

---

### 2.
今日は早く＿＿＿つもりです。  
✔️ **a) 帰る**

:::note Docente
Si el estudiante elige pasado o forma て,
el problema es **estructural**, no de vocabulario.
:::

---

## Parte 2 – Intención vs futuro

Frases:
- A) 明日、勉強します。  
- B) 明日、勉強するつもりです。

Respuesta correcta:
✔️ **B expresa decisión previa**

Explicación:
- A puede ser una declaración espontánea
- B implica que el plan ya estaba decidido

---

## Parte 3 – Negación de intención

### 4.
今日は外で＿＿＿つもりはありません。  
✔️ **a) 食べる**

Explicación:
- つもりはありません se construye con forma diccionario
- Expresa intención negativa clara

---

## Parte 4 – Comprensión funcional

Texto:
> 仕事が忙しいので、今週はあまり出かけないつもりです。  
> でも、週末は少し休もうと思っています。

Respuesta esperada:
- Durante la semana: no salir mucho
- Fin de semana: descansar un poco

:::warning
Si el estudiante no distingue los dos momentos,
falla en **seguimiento temporal**, no en gramática.
:::

---

## Parte 5 – Producción controlada

### Ejemplo válido (uno de muchos):

> 今週は忙しいので、あまり出かけないつもりです。  
> でも、週末は映画を見ようと思っています。  
> 日本語の勉強も続けるつもりです。

:::info Criterio
Se evalúa:
- uso correcto de estructuras
- coherencia de planes
- naturalidad N4
:::

---

## Evaluación global del Módulo 05

:::info Resultado
✔️ Aprobado si:
- つもりです se usa como intención real
- se distingue de futuro simple
- se interpreta contexto temporal
- la producción es clara y natural

❌ No aprobado si:
- se usa como tiempo verbal
- se mezclan formas incorrectas
- el discurso es confuso
:::

---

:::note Cierre docente
Este módulo define si el estudiante
**piensa en japonés al planificar**.

Sin este control,
el N4 queda incompleto.
:::
