# Evaluaciones del JLPT N4 – Dominio funcional

:::info Propósito académico
Esta sección **no evalúa memorización**.
Evalúa si el japonés N4:
- se comprende sin traducción palabra por palabra
- se produce con criterio
- se interpreta con lógica japonesa
- se sostiene bajo presión (simulacro)

Un N4 que no supera estas evaluaciones **no está cerrado**.
:::

---

## Qué evalúan estas pruebas

Cada evaluación mide:

- gramática en contexto
- comprensión de intención
- elección correcta de estructuras similares
- lectura funcional
- coherencia global del nivel

No se evalúa:
❌ velocidad  
❌ trucos  
❌ adivinanza  

Se evalúa:
✔️ comprensión real  
✔️ control estructural  
✔️ criterio lingüístico  

---

## Organización de las evaluaciones

:::info Estructura
:::

### Por módulo (01–06)
Cada módulo tiene:
- Examen
- Examen resuelto (criterio explicado)

### Simulacro final
- Examen completo estilo JLPT N4
- Resolución comentada (no solo respuestas)

---

## Regla premium de uso

:::warning Regla no negociable
No corrijas mirando la respuesta.
Primero **decide**, luego **justifica**, recién después **verifica**.
:::

Si fallas:
- anota **por qué**
- identifica si fue:
  - gramatical
  - cultural
  - de lectura
  - de prisa

---

:::note Nota docente final
Estas evaluaciones son el filtro que convierte
“estudié N4” en “domino N4”.

Sin superarlas con criterio,
el nivel no se considera cerrado.
:::
