# Forma て — Checklist de dominio real

:::info Propósito académico
Este checklist no mide “si viste el tema”.
Mide si **puedes usar la forma て correctamente sin traducir mentalmente**.
:::

La forma て es un **pilar estructural** del japonés N4.
Si este checklist no se cumple, avanzar genera errores acumulativos
en permiso, prohibición, peticiones y lectura real.

---

## 1. Dominio estructural básico

Marca solo si puedes hacerlo **de manera automática**:

- [ ] Conjugo correctamente verbos a forma て (sin tabla delante)
- [ ] Sé que la forma て **no cierra la oración**
- [ ] Cierro siempre con un verbo en forma final correcta
- [ ] No mezclo tiempos verbales incorrectamente

:::note Nota docente
Si dudas al conjugar → aún no hay automatización.
:::

---

## 2. Uso funcional correcto

Marca solo si eliges conscientemente la estructura adecuada:

- [ ] Uso forma て para **secuencia de acciones**
- [ ] Uso forma て para **contexto compartido**
- [ ] NO uso forma て para explicar causas
- [ ] Elijo から / ので cuando hay razón explícita

:::warning Error frecuente
Si usas forma て como “porque”, este bloque NO está dominado.
:::

---

## 3. Claridad y naturalidad

Marca solo si tus frases suenan naturales incluso al releerlas:

- [ ] No encadeno más de 2–3 acciones sin separar frases
- [ ] Mis secuencias son claras y fáciles de seguir
- [ ] Cada acción pertenece a la misma escena lógica
- [ ] Evito estructuras largas innecesarias

:::info Criterio japonés
Claridad > longitud  
Naturalidad > traducción literal
:::

---

## 4. Comprensión en lectura y escucha

Marca solo si puedes hacerlo sin detenerte demasiado:

- [ ] Reconozco secuencias con forma て al leer
- [ ] Entiendo el orden lógico de acciones en textos N4
- [ ] No interpreto forma て como causa en listening
- [ ] Identifico correctamente el verbo final de la oración

---

## 5. Producción autónoma

Marca solo si puedes producir sin copiar modelos:

- [ ] Escribo 5–8 frases con forma て sin errores
- [ ] Corrijo mis propios errores al releer
- [ ] Puedo explicar por qué usé forma て y no otra estructura

:::danger Advertencia docente
Si no puedes explicar tu elección,
es probable que haya sido intuitiva, no consciente.
:::

---

## 6. Resultado del checklist

### Interpretación

- ✔️ **0–1 fallos** → dominio sólido, puedes avanzar  
- ⚠️ **2–3 fallos** → repaso recomendado  
- ❌ **4 o más fallos** → NO avanzar, revisar desde introducción  

---

## 7. Acción obligatoria si fallas

:::note Plan de corrección
Si fallas este checklist:
1. Releer:
   - introducción a forma て
   - usos básicos
   - errores típicos
2. Crear 5 tarjetas Anki:
   - error incorrecto → corrección correcta
3. Reescribir 5 frases propias
:::
