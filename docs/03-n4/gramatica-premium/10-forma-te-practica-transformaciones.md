# Forma て — Práctica guiada de transformaciones

:::info Propósito académico
Esta práctica está diseñada para **automatizar la forma て**.
No es para “pensar la regla”, sino para **ejecutarla correctamente**.
:::

Si esta sección se siente lenta o confusa, **no avances**: la base aún no está consolidada.

---

## 1. Instrucciones generales (leer antes de empezar)

:::note Cómo trabajar correctamente
- No mires tablas mientras respondes.
- Escribe primero la respuesta.
- Corrige **solo al final**.
- Si fallas una misma transformación 2 veces → Anki obligatorio.
:::

---

## 2. Transformaciones básicas (presente → forma て)

Transforma los verbos a **forma て**.

1. 食べます →  
2. 行きます →  
3. 見ます →  
4. 飲みます →  
5. 話します →  
6. 書きます →  
7. 読みます →  
8. 来ます →  
9. します →  

:::warning Atención
- Verifica **terminación correcta**
- No mezcles て con た
:::

---

## 3. Transformaciones en contexto (obligatorio)

Reescribe las frases usando **forma て** para unir acciones.

### Ejemplo
> 家に帰りました。ごはんを食べました。  
→ 家に帰って、ごはんを食べました。

---

### Ejercicios

1. 朝起きました。学校に行きました。  
2. スーパーに行きました。パンを買いました。  
3. 友だちに会いました。映画を見ました。  
4. シャワーを浴びました。寝ました。  

:::info Criterio docente
Las acciones deben:
- pertenecer a la **misma escena**
- tener **orden lógico**
:::

---

## 4. Detección de errores (nivel examen)

Identifica y corrige el error.

1. ❌ 学校に行って、勉強しますでした。  
2. ❌ パンを食べて、コーヒーを飲むました。  
3. ❌ 友だちに会って、楽しいです。  

:::note Pista
La forma て **no determina el tiempo**.
El verbo final lo hace.
:::

---

## 5. Producción libre controlada

Escribe **5 frases propias** usando:

- forma て
- verbos conocidos
- una sola escena por frase

Requisitos:
- máximo 2 acciones por frase
- verbo final correcto
- sin traducción literal del español

---

## 6. Autoevaluación rápida

Marca solo si es cierto:

- [ ] Conjugo sin consultar tabla
- [ ] Uso forma て solo para secuencia/contexto
- [ ] No cierro la frase con forma て
- [ ] Identifico mis errores al releer

Si fallan 2 o más:
→ volver a ejercicios 2 y 3.
