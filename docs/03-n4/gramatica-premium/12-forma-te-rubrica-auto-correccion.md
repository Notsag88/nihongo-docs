# Forma て — Rúbrica de auto-corrección

:::info Propósito académico
Esta rúbrica te permite **evaluar objetivamente** tu uso de la forma て,
siguiendo criterios equivalentes a los que se aplican en:
- correcciones docentes
- simulacros JLPT N4
- revisión de producción escrita real
:::

No es un “checklist blando”.  
Es un **instrumento de control de calidad**.

---

## 1. Cómo usar esta rúbrica (método correcto)

Aplica esta rúbrica **después** de escribir:

- prácticas guiadas
- respuestas de examen
- frases propias

Procedimiento:
1. Relee tus frases en frío
2. Evalúa cada criterio
3. Marca **Sí / Parcial / No**
4. Corrige antes de avanzar

---

## 2. Rúbrica principal (forma て)

### Criterio A — Función correcta de la forma て

| Nivel | Descripción |
|------|-------------|
| ✔️ Sí | Uso て solo para **conectar acciones** |
| ⚠️ Parcial | A veces conecto sin relación clara |
| ❌ No | Uso て como cierre de frase |

Ejemplo incorrecto:
- ❌ 日本語を勉強して。  

Corrección:
- ✔️ 日本語を勉強して、上手になりました。

---

### Criterio B — Verbo final correcto

| Nivel | Descripción |
|------|-------------|
| ✔️ Sí | El **último verbo** marca tiempo y modalidad |
| ⚠️ Parcial | El tiempo es confuso |
| ❌ No | La frase termina en て |

Regla:
> La forma て **nunca** cierra la oración.

---

### Criterio C — Coherencia temporal

| Nivel | Descripción |
|------|-------------|
| ✔️ Sí | Todas las acciones encajan en el mismo marco temporal |
| ⚠️ Parcial | Alguna mezcla confusa |
| ❌ No | Presente + pasado sin justificación |

Ejemplo incorrecto:
- ❌ 昨日、家に帰って、寝ます。

Corrección:
- ✔️ 昨日、家に帰って、寝ました。

---

### Criterio D — Relación semántica entre acciones

| Nivel | Descripción |
|------|-------------|
| ✔️ Sí | Las acciones tienen relación lógica |
| ⚠️ Parcial | Relación débil |
| ❌ No | Acciones inconexas |

Ejemplo incorrecto:
- ❌ パンを食べて、雨が降りました。

---

### Criterio E — Complejidad adecuada a N4

| Nivel | Descripción |
|------|-------------|
| ✔️ Sí | 2–3 acciones claras |
| ⚠️ Parcial | Demasiadas acciones |
| ❌ No | Frases largas y confusas |

:::note Regla docente
Más complejidad **no** significa mejor japonés.
:::

---

## 3. Puntuación recomendada

Asigna:
- ✔️ Sí = 2 puntos
- ⚠️ Parcial = 1 punto
- ❌ No = 0 puntos

### Interpretación

| Puntaje | Nivel |
|------|------|
| 8–10 | Dominio adecuado |
| 5–7 | Aceptable, requiere refuerzo |
| ≤ 4 | No avanzar |

---

## 4. Errores críticos (bloqueantes)

:::danger Bloqueo de avance
Si aparece **cualquiera** de estos errores, no avances:
:::

- cerrar frases con て
- usar て como “y” español sin lógica
- mezclar tiempos sin control
- repetir el mismo verbo sin razón

Estos errores se **fijan** si no se corrigen ahora.

---

Si el resultado es bajo:

---

## 6. Criterio de cierre del bloque Forma て

Puedes considerar **cerrado** este bloque cuando:

- [ ] No terminas frases en て
- [ ] El tiempo siempre está claro
- [ ] Conectas acciones con lógica
- [ ] Reconoces errores al leerlos
