:::info
### Núcleo del contenido

En este bloque comienza el estudio **estructural y académico** de las **comparaciones en japonés** para nivel **JLPT N4**.

El objetivo **no** es aprender “formas para comparar”, sino **comprender cómo el japonés organiza la comparación**, qué elementos compara, y **desde qué punto de vista**.

Este contenido es **fundacional** para:
- expresar preferencias
- comparar opciones
- justificar elecciones
- preparar comparaciones complejas de N3
:::

---

## 1. Qué significa comparar en japonés

:::note
### Enfoque docente nativo

En español solemos comparar diciendo directamente:

> “A es más X que B.”

En japonés, la comparación **no es simétrica**.  
Siempre se define **un punto de referencia**, y desde allí se evalúa.

Comparar en japonés implica responder a la pregunta:

> **¿Desde qué elemento estoy hablando?**
:::

---

## 2. Idea central del bloque de comparaciones

:::info
### Principio estructural japonés

El japonés compara usando estas ideas clave:

- hay un **referente**
- hay un **elemento evaluado**
- hay una **propiedad comparada**

No se comparan “dos cosas en igualdad”,  
se evalúa **una cosa respecto de otra**.
:::

---

## 3. Formas principales de comparación en N4

En este nivel se trabajan tres estructuras **canónicas**:

| Estructura | Función | Estado |
|---|---|---|
| 〜より〜ほうが | Comparación directa | Fundamental |
| 〜のほうが | Preferencia | Fundamental |
| どちらが〜 | Elección entre dos | Fundamental |

:::warning
Estas estructuras **no son intercambiables**  
y cada una responde a una intención distinta.
:::

---

## 4. Comparación ≠ enumeración

:::warning
### Error conceptual frecuente

❌ Pensar que comparar es listar características.

Ejemplo incorrecto:
- Aは大きいです。Bは小さいです。

Esto **no es una comparación estructural**.

✔️ Comparación japonesa:
- BよりAのほうが大きいです。
:::

---

## 5. Comparar implica punto de vista

:::note
### Idea clave para todo el bloque

Cuando dices:

- 日本はフランスより安全です。

Estás diciendo:
> “Desde Francia como referencia, Japón es más seguro.”

El **orden importa**, porque define el punto de vista.
:::

---

## 6. Comparaciones y subjetividad

:::info
### Nivel N4

En N4:
- las comparaciones pueden ser **objetivas** (precio, tamaño)
- o **subjetivas** (gustos, comodidad)

Pero **la estructura gramatical no cambia**,  
solo cambia la **intención del hablante**.
:::

---

## 7. Qué NO se espera todavía (límite N4)

:::note
### Delimitación académica

En N4 **no se exige**:
- comparar más de dos elementos
- comparaciones abstractas complejas
- argumentación extensa

Eso pertenece a **N3**.

Aquí se exige:
- estructura correcta
- orden natural
- claridad del punto de vista
:::

## 9. Cierre docente

:::note
Comparar en japonés **no es traducir “más que”**.

Es:
- definir un referente
- adoptar un punto de vista
- evaluar con estructura clara

Este bloque marca un salto importante en **precisión expresiva**.
