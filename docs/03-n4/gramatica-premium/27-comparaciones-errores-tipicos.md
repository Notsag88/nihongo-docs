---
title: Comparaciones en japonés N4 – Errores típicos y correcciones canónicas
---

:::info
### Núcleo del archivo

Este documento identifica y corrige los **errores más frecuentes y persistentes** en el uso de **comparaciones en japonés (N4)**, especialmente con:

- **〜より〜ほうが**
- **どちらが**
- comparaciones con verbos
- comparaciones + opinión (**〜と思います**)

Muchos de estos errores producen japonés:
- gramaticalmente aceptable
- pero **conceptualmente incorrecto o antinatural**

Este archivo cumple una función **correctiva y de control de calidad**.
:::

---

## 1. Error 1 — Traducir literalmente “más que”

:::warning
### Error conceptual básico

❌ Pensar:
> “A es más X que B” → AよりBのほうがX

Esto invierte el **punto de vista japonés**.
:::

### Forma correcta

- ✔️ BよりAのほうがXです。

Ejemplo:

- ❌ 日本より私の国のほうが安全です。  
- ✔️ 私の国より日本のほうが安全です。

:::note
En japonés, **より marca el referente**, no el foco.
:::

---

## 2. Error 2 — Invertir より y ほうが

:::warning
### Error estructural grave

❌ AのほうがBよりXです。

Esta estructura **no existe** en japonés estándar.
:::

### Regla obligatoria

> **Bより Aのほうが X**

Siempre en ese orden.
:::

---

## 3. Error 3 — Comparar sin criterio explícito

:::warning
### Error frecuente con どちらが

❌ AとBと、どちらがですか。  
(pregunta incompleta)

### Forma correcta

- ✔️ AとBと、どちらが**高い**ですか。  
- ✔️ AとBと、どちらが**好き**ですか。

:::note
Comparar siempre requiere un **criterio visible**.
:::

---

## 4. Error 4 — Responder どちらが sin 〜のほうが

:::warning
❌ 電車です。  
(respuesta incompleta)

### Forma natural

- ✔️ 電車のほうが早いです。

:::note
La respuesta **retoma la estructura comparativa**.
:::

---

## 5. Error 5 — Omitir と思います en comparaciones subjetivas

:::warning
### Error pragmático

❌ 電車よりバスのほうが便利です。  
(usado como opinión personal)

### Corrección

- ✔️ 電車よりバスのほうが便利だと思います。

:::note
En juicios personales, **marcar opinión es preferible**.
:::

---

## 6. Error 6 — Usar comparaciones para imponer

:::warning
❌ 電車よりバスのほうがいいですから、これにしなさい。  
(tono impositivo, poco natural)

### Forma natural

- ✔️ 電車よりバスのほうがいいと思います。

:::note
Las comparaciones **sugieren**, no ordenan.
:::

---

## 7. Error 7 — Comparar más de dos elementos con どちらが

:::warning
❌ AとBとCと、どちらがいいですか。

### Corrección

- ✔️ AとBと、どちらがいいですか。  
- (tres opciones pertenece a N3)

:::note
**どちらが = exactamente dos opciones**.
:::

---

## 8. Error 8 — Comparaciones con forma て mal usadas

:::warning
❌ 電車が速くて、バスよりいいです。  
(estructura híbrida poco clara)

### Corrección

- ✔️ 電車よりバスのほうが遅いです。  
- ✔️ 電車は速いですから、便利だと思います。

:::note
La forma て **no sustituye** a la estructura comparativa.
:::

---

## 9. Resumen de errores críticos (tabla)

| Error | Problema | Corrección |
|---|---|---|
| Traducción literal | Punto de vista incorrecto | Pensar referente |
| Orden invertido | Estructura inexistente | BよりAのほうが |
| Sin criterio | Comparación incompleta | Añadir X |
| Respuesta corta | Falta estructura | Usar のほうが |
| Opinión sin marca | Ambigüedad | Añadir と思います |
| Comparar imponiendo | Tono incorrecto | Suavizar |
| Tres opciones | Uso indebido | Limitar a dos |
| Forma て híbrida | Ambigüedad | Separar funciones |

---

## 10. Cierre docente

:::info
Dominar estos errores implica:

- producir comparaciones **claras y naturales**
- evitar japonés “traducido”
- mantener control del punto de vista
- preparar el terreno para:
  - comparaciones múltiples (N3)
  - argumentación comparativa

Este archivo debe revisarse **antes de cerrar el bloque de comparaciones**.
:::
