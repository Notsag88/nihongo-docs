---
title: Experiencia en japonés N4 – Errores típicos y correcciones canónicas
---

:::info
### Núcleo del archivo

Este documento identifica y corrige los **errores más frecuentes y persistentes** relacionados con la **expresión de experiencia y frecuencia** en japonés nivel **JLPT N4**, especialmente con:

- **〜たことがある**
- pasado puntual (**〜た / 〜ました**)
- frecuencia (**回 / いつも / よく / ときどき**)

Muchos de estos errores producen japonés:
- gramaticalmente aceptable
- pero **conceptualmente incorrecto o antinatural**

Este archivo cumple una función **correctiva y estabilizadora**.
:::

---

## 1. Error 1 — Usar pasado puntual para hablar de experiencia

:::warning
### Error conceptual crítico

❌ 日本に行きました。  
(usado para decir “he ido a Japón alguna vez”)

### Corrección

- ✔️ 日本に行ったことがあります。

:::note
El pasado puntual **no expresa experiencia de vida**.
:::

---

## 2. Error 2 — Usar 〜たことがある con fechas concretas

:::warning
### Error grave de incompatibilidad

❌ 去年、日本に行ったことがあります。  
❌ 昨日、寿司を食べたことがあります。

### Corrección

- ✔️ 去年、日本に行きました。  
- ✔️ 寿司を食べたことがあります。

:::note
La experiencia **no admite tiempo concreto**.
:::

---

## 3. Error 3 — Preguntar “cuándo” con experiencia

:::warning
### Error estructural

❌ いつ日本に行ったことがありますか。

### Corrección

- ✔️ 日本に行ったことがありますか。  
- ✔️ いつ日本に行きましたか。

:::note
“Cuándo” y experiencia **no se combinan**.
:::

---

## 4. Error 4 — Confundir experiencia con hábito

:::warning
### Error de concepto

❌ 日本語を勉強したことがあります。  
(usado para decir “estudio japonés habitualmente”)

### Corrección

- ✔️ 日本語を勉強しています。  
- ✔️ よく日本語を勉強します。

:::note
La experiencia **no expresa hábitos actuales**.
:::

---

## 5. Error 5 — Usar adverbios de frecuencia con experiencia sin control

:::warning
### Error frecuente

❌ よく日本に行ったことがあります。

### Corrección

- ✔️ 日本に行ったことがあります。  
- ✔️ 去年、日本によく行きました。

:::note
“よく” expresa **repetición**, no experiencia general.
:::

---

## 6. Error 6 — Olvidar la forma negativa correcta (nunca)

:::warning
### Error estructural

❌ 日本に行ったことがないです。  
(forma informal / inestable)

### Corrección canónica N4

- ✔️ 日本に行ったことがありません。  
- ✔️ 日本に一度も行ったことがありません。

:::note
La forma negativa **se fija en ありません**.
:::

---

## 7. Error 7 — Confundir frecuencia puntual con experiencia

:::warning
### Error de mezcla conceptual

❌ 日本に三回行ったことがあります。  
(usado con referencia temporal implícita)

### Corrección

- ✔️ 日本に行ったことがあります。  
- ✔️ 日本に三回行きました。  

:::note
Decide primero: **experiencia** o **evento contado**.
:::

---

## 8. Error 8 — Respuestas innecesariamente largas

:::warning
### Error pragmático

❌ はい、日本に行ったことがあります。去年行きました。  
(mezcla sin transición)

### Forma más natural

- ✔️ はい、日本に行ったことがあります。  
- ✔️ 去年、日本に行きました。

:::note
Separar experiencia y evento **mejora claridad**.
:::

---

## 9. Resumen de errores críticos (tabla)

| Error | Problema | Corrección |
|---|---|---|
| Pasado por experiencia | Concepto incorrecto | Usar 〜たことがある |
| Fecha + experiencia | Incompatibilidad | Usar pasado |
| “Cuándo” + experiencia | Estructura rota | Separar |
| Experiencia por hábito | Significado errado | Usar presente |
| Frecuencia + experiencia | Mezcla conceptual | Elegir forma |
| Negativo incorrecto | Forma inestable | ありません |
| Conteo mal usado | Ambigüedad | Decidir concepto |
| Respuestas largas | Falta claridad | Separar ideas |

---

## 10. Cierre docente

:::info
Dominar estos errores implica:

- usar la experiencia con precisión
- evitar ambigüedades frecuentes en N4
- producir japonés **conceptualmente correcto**
- preparar el terreno para:
  - experiencia evaluada (N3)
  - narración más rica

Este archivo debe revisarse **antes de cerrar el bloque de experiencia**.
:::
