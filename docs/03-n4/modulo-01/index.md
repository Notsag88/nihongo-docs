# Módulo 01 – Integración inicial del N4

:::info Propósito académico
Este módulo marca el **inicio operativo del nivel JLPT N4**.

Su función no es introducir contenido nuevo, sino **integrar de forma funcional** lo ya trabajado en:
- **Gramática Premium** (forma て)
- **Kanji Premium** (núcleo N4)
- **Lecturas Premium** (rutina y contexto real)

Aquí comienza la transición de:
❌ estudio por bloques  
a  
✔️ uso integrado del idioma
:::

---

## Rol del módulo dentro del nivel N4

:::note Visión docente
Un módulo N4 no es un “capítulo del libro”.

Es una **unidad de entrenamiento lingüístico**, donde el estudiante debe:
- reconocer
- usar
- cometer errores
- corregirlos
:::

Este módulo trabaja especialmente:
- conexión de acciones
- rutinas
- descripción básica de actividades
- lectura y producción controlada

---

## Ejes lingüísticos del Módulo 01

:::info Contenidos integrados
Este módulo articula los siguientes ejes:
:::

### 1. Gramática principal
- Forma て:
  - secuencia de acciones
  - rutinas
  - descripciones funcionales

### 2. Vocabulario
- actividades diarias
- estudio y trabajo
- acciones frecuentes

### 3. Kanji
- kanji de uso cotidiano
- verbos y sustantivos de alta frecuencia
- reconocimiento en contexto

### 4. Lectura
- textos cortos sobre vida diaria
- lectura por bloques de sentido
- integración de gramática y kanji

## Cómo estudiar este módulo (regla premium)

:::warning Regla no negociable
No avances al siguiente archivo “porque toca”.

Avanza solo cuando puedas:
- reconocer las estructuras sin pensar
- leer frases completas sin traducir todo
- producir ejemplos simples sin copiar
:::

Metodología sugerida:
1. Gramática aplicada

## Criterio de dominio del módulo

Antes de considerar este módulo como “cerrado”, deberías poder:

- [ ] Usar la forma て para conectar acciones
- [ ] Leer frases con forma て sin analizarlas
- [ ] Reconocer kanji frecuentes en rutinas
- [ ] Producir descripciones simples de tu día
- [ ] Completar el checkpoint con seguridad

Si esto no se cumple:
→ volver a este módulo es **correcto y necesario**.
