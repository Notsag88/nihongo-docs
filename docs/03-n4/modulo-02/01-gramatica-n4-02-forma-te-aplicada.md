# Gramática N4 — Forma て aplicada (permiso, prohibición, peticiones)

:::info Propósito académico
Este archivo operacionaliza la forma て en **interacciones reales**.  
No reenseña teoría: entrena **uso automático** en cuatro funciones N4 de alta frecuencia:

- pedir permiso (〜てもいいですか)
- dar permiso (〜てもいいです)
- prohibir (〜てはいけません)
- pedir (〜てください／〜ないでください)

El objetivo es que puedas **actuar** en japonés, no solo “saber la forma”.
:::

---

## 1) Permiso: 〜てもいいですか（¿puedo…?)

:::info Función
Pedir autorización de forma neutra/cortés.
:::

Estructura:
- Vて + もいいですか

Ejemplos:
- ここで写真を撮ってもいいですか。  
- この席に座ってもいいですか。  
- 窓を開けてもいいですか。

:::warning Error frecuente
Confundir permiso con capacidad.
- 〜てもいいですか = autorización
- 〜できますか = capacidad / posibilidad
:::

---

## 2) Dar permiso: 〜てもいいです（sí, puedes)

:::info Función
Autorizar o permitir.
:::

Ejemplos:
- はい、撮ってもいいですよ。  
- いいですよ。  
- ええ、どうぞ。

:::note Matiz docente
“どうぞ” funciona como invitación/permiso amable.  
No es “sí” literal, es habilitar la acción.
:::

---

## 3) Prohibición: 〜てはいけません（no se permite)

:::info Función
Regla, norma, prohibición institucional o moral.
:::

Estructura:
- Vて + はいけません

Ejemplos:
- ここでタバコを吸ってはいけません。  
- この部屋に入ってはいけません。  
- 授業中に話してはいけません。

:::warning Error frecuente
Suavizar demasiado cuando el contexto es norma.
En carteles y reglas, 〜てはいけません es apropiado.
:::

---

## 4) Petición estándar: 〜てください（por favor, haz…)

:::info Función
Pedido directo y cortés (estándar N4).
:::

Ejemplos:
- ここに名前を書いてください。  
- ちょっと待ってください。  
- もう一度言ってください。

:::note Matiz docente
En situaciones reales, 〜てください es “lo normal”.  
Si intentas “ser demasiado formal” sin base, suena artificial.
:::

---

## 5) Petición negativa: 〜ないでください（por favor, no hagas…)

:::info Función
Pedir que no se realice una acción (cortés estándar).
:::

Ejemplos:
- ここで食べないでください。  
- まだ行かないでください。  
- 大きい声で話さないでください。

:::warning Error frecuente
Confundir con prohibición institucional.
- 〜ないでください = petición al interlocutor
- 〜てはいけません = regla / norma
:::

---

## 6) Respuestas típicas (modelo realista)

:::info Respuestas al permiso
- はい、いいですよ。  
- いいですよ。  
- すみません、ちょっと…（rechazo suave)
:::

:::info Respuestas a una petición
- はい、わかりました。  
- いいですよ。  
- すみません、今はちょっと…（no puedo ahora)
:::

---

## 7) Integración con 〜てから (procedimientos)

:::note Uso natural en instrucciones
- Vてから + acción principal
:::

Ejemplos:
- ここに名前を書いてから、受付へ行ってください。  
- チケットを買ってから、入ってください。

---

## 8) Indicadores de dominio en este módulo

:::info
Esta gramática está integrada si:
- produces permiso/prohibición/petición sin traducir desde español
- entiendes carteles simples sin “analizar”
- respondes de forma corta y natural (いいですよ／わかりました)
:::

---

:::note Cierre docente
En N4, la forma て no es solo “conectar acciones”.
Es la base de cómo se negocia conducta, cortesía y normas en japonés.
:::
