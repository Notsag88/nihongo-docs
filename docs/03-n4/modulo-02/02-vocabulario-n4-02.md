# Vocabulario N4 — Módulo 02

:::info Propósito académico
Este vocabulario está seleccionado para **hacer operativa** la gramática del módulo:

- permiso (〜てもいいですか)
- prohibición (〜てはいけません)
- peticiones (〜てください／〜ないでください)
- instrucciones simples (〜てから…)

No es una lista “para memorizar”: es un conjunto de **unidades de uso**.
:::

---

## 1) Verbos funcionales de alta frecuencia (para pedidos y normas)

:::info
Aprende estos verbos como bloques con forma て / forma diccionario:
:::

| Verbo | Lectura | Uso típico |
|---|---|---|
| 使う | つかう | usar (algo) |
| 入る | はいる | entrar |
| 出る | でる | salir |
| 座る | すわる | sentarse |
| 立つ | たつ | ponerse de pie |
| 待つ | まつ | esperar |
| 書く | かく | escribir |
| 開ける | あける | abrir |
| 閉める | しめる | cerrar |
| 話す | はなす | hablar |
| 写真を撮る | しゃしんをとる | sacar foto |

:::note
Estos verbos sostienen el 80% de los ejemplos de permiso/pedido en N4.
:::

## 3) Expresiones de cortesía mínima (respuesta realista)

:::info Respuestas cortas naturales
- はい（sí)
- いいですよ（está bien)
- どうぞ（adelante)
- わかりました（entendido)
- すみません（disculpa)
- ちょっと…（rechazo suave / “mmm…”)
:::

:::warning
“ちょっと…” es muy japonés: evita decir “no” frontal.
No es “un poquito” literal.
:::

---

## 4) Bloques listos para uso (patrones)

:::info Permiso
- 〜てもいいですか  
- ここで＿＿＿てもいいですか  
- ＿＿＿てもいいですか
:::

:::info Prohibición / normas
- 〜てはいけません  
- ここで＿＿＿てはいけません
:::

:::info Peticiones
- 〜てください  
- 〜ないでください
:::

---

## 5) Señales y mensajes típicos (lectura rápida)

:::note
No memorices traducción palabra por palabra.
Asocia forma → función:
:::

- 入ってください。  
- ここで食べないでください。  
- 写真を撮ってはいけません。  
- 入口はこちらです。  
- 受付で聞いてください。

---

## 6) Errores típicos con vocabulario del módulo

:::warning
- Usar “できますか” cuando corresponde permiso (〜てもいいですか)
- Pedir “no hagas” con 〜てはいけません en contexto personal (mejor 〜ないでください)
- Responder demasiado largo: en japonés muchas respuestas correctas son breves
:::

---

## 7) Recomendación Anki (criterio premium)

:::info
Tarjetas de alto retorno:
- Frente: bloque (例：写真を撮ってもいいですか)
- Reverso: lectura + significado operativo breve
- Variantes: cambiar objeto/lugar
:::

:::warning
Evitar tarjetas sin contexto (solo “入口 = entrada”).
Mejor: 入口はこちらです。
:::

---

:::note Cierre docente
El vocabulario de este módulo no se mide por cantidad,
sino por tu capacidad de **interactuar**: pedir, responder y comprender normas.
:::
