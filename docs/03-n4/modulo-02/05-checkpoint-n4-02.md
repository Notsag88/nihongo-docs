# Checkpoint N4 — Módulo 02

:::info Propósito académico
Este checkpoint verifica si el Módulo 02 fue **integrado funcionalmente**:

- permiso (〜てもいいですか／〜てもいいです)
- prohibición (〜てはいけません)
- peticiones (〜てください／〜ないでください)
- lectura de avisos simples
- respuestas naturales mínimas

La pregunta central es:

> **¿Puedo interactuar en japonés N4 sin traducir cada paso?**
:::

---

## 1) Dominio conceptual (base)

Marca solo si puedes explicarlo:

- ⬜ Permiso (〜てもいい) ≠ capacidad (〜できる)
- ⬜ Prohibición (〜てはいけません) ≠ petición negativa (〜ないでください)
- ⬜ 〜てください es la petición estándar del módulo
- ⬜ En lectura, estas formas deben reconocerse sin análisis

---

## 2) Reconocimiento rápido (función)

Identifica mentalmente:

1. 入ってもいいですか。 → ⬜ permiso  
2. ここで話してはいけません。 → ⬜ prohibición  
3. ちょっと待ってください。 → ⬜ petición  
4. ここで食べないでください。 → ⬜ petición negativa  

Marca:
- ⬜ Las identifiqué sin dudar

---

## 3) Producción controlada (permiso)

Produce 2 frases:

- ＿＿＿＿てもいいですか。  
- ＿＿＿＿てもいいですか。

Requisito:
- verbos del módulo (使う／入る／座る／写真を撮る etc.)

Autoevaluación:
- ⬜ Son naturales
- ⬜ No usé できますか cuando era permiso

---

## 4) Producción controlada (prohibición / petición negativa)

Escribe una prohibición tipo “regla”:
- ＿＿＿＿てはいけません。

Escribe una petición negativa (a una persona):
- ＿＿＿＿ないでください。

Autoevaluación:
- ⬜ Distinguí bien el contexto
- ⬜ Las formas son correctas

---

## 5) Lectura de avisos (flujo)

Lee una vez:

> 入口はこちらです。出口はあちらです。  
> 受付で名前を書いてください。  
> ここで写真を撮ってはいけません。

Marca:
- ⬜ Entendí el procedimiento general
- ⬜ Reconocí 入口／出口／受付 sin detenerme
- ⬜ Identifiqué claramente lo prohibido

---

## 6) Respuestas naturales mínimas

Responde como B:

A: ここで座ってもいいですか。  
B: ＿＿＿＿＿＿＿＿

Opciones válidas:
- いいですよ。  
- はい、どうぞ。  
- すみません、ちょっと…

Marca:
- ⬜ Respondí breve y natural (sin explicaciones largas)

---

## 7) Evaluación global del módulo

:::info Evaluación final
- ⬜ Integración sólida (puedo usarlo en contexto)
- ⬜ Integración parcial (necesito repetir práctica)
- ⬜ No integrado (debo volver a gramática aplicada)
:::

:::note Cierre académico
Si el módulo es sólido:
✔️ Módulo 02 queda formalmente cerrado  
✔️ Puedes avanzar a Módulo 03 con estabilidad

Si es parcial:
→ repetir práctica guiada es una decisión correcta.
:::
