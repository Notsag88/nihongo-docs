# Módulo 02 – Forma て aplicada (interacciones y normas)

:::info Propósito académico
Este módulo convierte la forma て en **herramienta social y funcional**.

Si el Módulo 01 entrenó la forma て para **rutinas y secuencias**, aquí el foco es:
- pedir
- permitir
- prohibir
- coordinar acciones
- seguir reglas y señales del entorno

Objetivo: que la forma て deje de ser “una estructura” y pase a ser **un recurso automático de interacción**.
:::

---

## Rol del módulo dentro del nivel N4

:::note Visión docente
En N4, el idioma empieza a entrar en situaciones reales:

- transporte
- tienda / recepción
- oficina / universidad
- espacios públicos con normas

La gramática no se “recuerda”: se **ejecuta** en contexto.
:::

Este módulo entrena especialmente:
- 〜てもいいですか／〜てもいいです
- 〜てはいけません
- 〜てください（petición estándar)
- 〜ないでください（petición negativa)
- secuencias funcionales con 〜てから／〜て

---

## Ejes lingüísticos del Módulo 02

:::info Contenidos integrados
:::

### 1) Gramática aplicada
- permiso / autorización
- prohibición / normas
- peticiones (positivas y negativas)
- secuencias con intención práctica

### 2) Vocabulario
- instrucciones y señales
- acciones típicas en espacios públicos
- cortesía básica para pedir y responder

### 3) Kanji
- kanji frecuentes en carteles y contexto (入口・出口・禁止, etc.)
- verbos de acción aplicables (使う・入る・出る, etc.)
- reconocimiento como prioridad

### 4) Lectura / microlectura
- diálogos cortos funcionales
- textos tipo “reglas / avisos”
- comprensión sin traducir palabra por palabra

## Criterio de dominio del módulo

Antes de cerrar el módulo deberías poder:

- [ ] Formular permiso: 〜てもいいですか
- [ ] Formular prohibición: 〜てはいけません
- [ ] Pedir de forma estándar: 〜てください
- [ ] Pedir que no se haga algo: 〜ないでください
- [ ] Entender reglas simples en carteles / avisos
- [ ] Responder de forma simple y natural (いいですよ／すみません、ちょっと…)
