# Vocabulario N4 — Módulo 03

:::info Propósito académico
Este vocabulario está seleccionado para producir frases con:
- から／ので (razones)
- と思います (opinión)

Criterio:
- palabras de alto retorno
- motivos cotidianos y universales
- combinables con verbos del módulo
:::

---

## 1) Motivos frecuentes (razones típicas N4)

| Motivo | Lectura | Uso típico |
|---|---|---|
| 時間がない | じかんがない | falta de tiempo |
| 忙しい | いそがしい | estar ocupado |
| 疲れた | つかれた | estar cansado |
| 用事がある | ようじがある | tener un asunto/mandado |
| お金がない | おかねがない | falta de dinero |
| 遠い | とおい | estar lejos |
| 近い | ちかい | estar cerca |
| 便利 | べんり | conveniente |
| 大切 | たいせつ | importante |

:::note
Estos motivos generan frases reales sin depender de vocabulario raro.
:::

---

## 3) Expresiones de evaluación (opinión breve)

:::info
Bloques útiles con と思います:
:::

- 〜のほうがいいと思います。  
- 〜が大切だと思います。  
- 〜は便利だと思います。  
- 〜は難しいと思います。  

:::warning
No forzar “opinión” en cada frase.
Úsala cuando sea cierre o matiz.
:::

---

## 4) Bloques listos para combinar (alto retorno)

:::info Razones
- 時間がないので、＿＿＿。  
- 用事があるので、＿＿＿。  
- 疲れたから、＿＿＿。  
- お金がないから、＿＿＿。
:::

:::info Opinión
- ＿＿＿と思います。  
- ＿＿＿と思っています。
:::

---

## 5) Microfrases modelo (no memorizar, usar como referencia)

:::note
- 時間がないので、今日は行けません。  
- 疲れたから、早く寝ます。  
- 日本語は難しいですが、大切だと思います。  
- この店は便利だと思います。
:::

---

## 6) Errores típicos con vocabulario del módulo

:::warning
- Usar razones demasiado largas (N4: mantener simple)
- Mezclar から／ので sin intención
- Repetir と思います como muletilla
:::
