# Módulo 03 – Razones y opinión (N4)

:::info Propósito académico
Este módulo entrena una habilidad que marca el salto real N5 → N4:

**dejar de decir hechos aislados**  
y empezar a **conectar ideas con razones** y **expresar postura**.

Aquí se integra:
- から / ので (razón)
- orden lógico básico
- opinión con 〜と思います／〜と思っています

Objetivo: producir japonés que suene a **pensamiento humano**, no a frases sueltas.
:::

---

## Rol del módulo dentro del nivel N4

:::note Visión docente
En japonés real, las razones se usan constantemente para:
- justificar decisiones
- explicar acciones
- suavizar afirmaciones
- responder “por qué” sin tensión

En N4, dominar から／ので no es “saber dos partículas”:  
es dominar **la lógica del discurso**.
:::

---

## Ejes lingüísticos del Módulo 03

:::info Contenidos integrados
:::

### 1) Gramática aplicada
- 〜から (razón más directa / personal)
- 〜ので (razón más neutral / suave)
- conectores de orden mínimo
- 〜と思います／〜と思っています (opinión / postura)

### 2) Vocabulario funcional
- motivos cotidianos: tiempo, dinero, cansancio, distancia, salud
- verbos de decisión: 行く／やめる／続ける／選ぶ
- expresiones de preferencia y evaluación simples

### 3) Kanji
- 理由・思・大切・便利・時間・お金・健康 など
- reconocimiento prioritario en lectura y producción mínima

### 4) Lectura / microlectura
- textos cortos de “decisión + razón”
- comprensión global sin traducción palabra por palabra

## Cómo estudiar este módulo (regla premium)

:::warning Regla no negociable
No memorices から y ので como “sinónimos”.
Debes aprender **cuándo elegir cada uno** y cómo suena en contexto.

La meta es:
- explicar acciones sin esfuerzo
- dar razones sin sonar brusco
- expresar opinión breve y natural
:::

---

## Criterio de dominio del módulo

Antes de cerrar el módulo deberías poder:

- [ ] Dar una razón simple con から
- [ ] Dar una razón simple con ので
- [ ] Elegir entre から／ので en contextos típicos
- [ ] Cerrar una idea con 〜と思います
- [ ] Leer textos cortos con razones sin perder el hilo
