# Vocabulario N4 — Módulo 04

:::info Propósito académico
Este vocabulario permite comparar de forma natural sin “inventar palabras”.

Se centra en:
- adjetivos de evaluación frecuentes
- sustantivos comparables del día a día
- bloques listos para usar con より〜ほうが／どちらが

Criterio: frecuencia + combinabilidad + retorno en lectura y examen.
:::

---

## 1) Adjetivos de comparación (alta prioridad)

| Adjetivo | Lectura | Par típico |
|---|---|---|
| 高い | たかい | 安い |
| 安い | やすい | 高い |
| 便利 | べんり | 不便 |
| 早い | はやい | 遅い |
| 遅い | おそい | 早い |
| 難しい | むずかしい | やさしい |
| やさしい | やさしい | 難しい |
| 分かりやすい | わかりやすい | 分かりにくい（reconocer) |

:::note
No necesitas todos los opuestos para empezar.
Prioriza los que aparecen en tus lecturas y módulos.
:::

---

## 3) Bloques listos para uso (patrones)

:::info Comparación
- AよりBのほうが＿＿＿です。  
- Bのほうが＿＿＿です。
:::

:::info Pregunta
- AとBと、どちらが＿＿＿ですか。  
- どちらのほうが＿＿＿ですか。
:::

:::info Conclusión con opinión
- ＿＿＿のほうがいいと思います。  
- ＿＿＿のほうが＿＿＿ので、＿＿＿と思います。
:::

---

## 4) Microfrases modelo (referencia, no para memorizar)

:::note
- バスより電車のほうが早いです。  
- この店のほうが安いです。  
- この方法のほうが分かりやすいと思います。  
- 東京と大阪と、どちらが便利ですか。
:::

---

## 5) Errores típicos con vocabulario de comparación

:::warning
- Comparar sin criterio (“todo es 便利”)
- Usar adjetivos demasiado abstractos sin control
- Hacer frases largas y perder naturalidad N4
:::

---

## 6) Recomendación Anki (criterio premium)

:::info
Tarjetas de alto retorno:
- Frente: patrón completo (AよりBのほうが〜)
- Reverso: lectura + significado operativo breve
- Variaciones: cambiar A/B y adjetivo
:::

:::warning
Evitar tarjetas de “adjetivo aislado”.
La comparación se aprende como estructura.
:::
