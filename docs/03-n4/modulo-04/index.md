# Módulo 04 – Comparaciones y elección (N4)

:::info Propósito académico
Este módulo entrena una capacidad central del N4:

- comparar opciones
- elegir con criterio
- justificar una preferencia de forma natural

Aquí se integra:
- より〜ほうが (comparación explícita)
- 〜のほうが〜 (preferencia)
- どちらが〜 (elección entre dos)
- conectores mínimos para sostener discurso (ですが／ので／と思います)

Objetivo: que puedas comparar sin traducir “más que” y sin sonar mecánico.
:::

---

## Rol del módulo dentro del nivel N4

:::note Visión docente
En N5 dices “me gusta A”.
En N4 empiezas a decir:

- “A es mejor que B”
- “prefiero A porque…”
- “¿cuál es más…?”

Esto aparece en:
- lectura
- escucha
- conversación real
- preguntas del JLPT

Comparar es un salto cognitivo: requiere **estructura lógica** y control de matices.
:::

---

## Ejes lingüísticos del Módulo 04

:::info Contenidos integrados
:::

### 1) Gramática aplicada
- より〜ほうが (AよりBのほうが…)
- 〜のほうが (preferencia sin comparar explícito)
- どちらが (pregunta de elección)
- integración con razón/opinión en nivel N4

### 2) Vocabulario funcional
- adjetivos frecuentes de evaluación:
  - 高い／安い
  - 便利／不便
  - 早い／遅い
  - 難しい／易しい（si aparece)
- sustantivos comparables:
  - 店／町／電車／バス／勉強方法 など

### 3) Kanji
- 高・安・便・利・早・遅 など
- reconocimiento en lectura como prioridad

### 4) Lectura / microlectura
- textos de “comparación + conclusión”
- evitar traducción palabra por palabra

## Cómo estudiar este módulo (regla premium)

:::warning Regla no negociable
No memorices patrones como fórmula vacía.

La comparación debe sonar como:
- observación real
- preferencia razonada
- elección natural

Si solo puedes “llenar huecos”, aún no lo dominas.
:::

---

## Criterio de dominio del módulo

Antes de cerrar este módulo deberías poder:

- [ ] Formar AよりBのほうが〜 con naturalidad
- [ ] Usar 〜のほうが para preferencia sin comparar explícito
- [ ] Preguntar con どちらが〜
- [ ] Integrar una razón simple (ので／から) en la comparación
- [ ] Leer comparaciones N4 sin analizar cada partícula
