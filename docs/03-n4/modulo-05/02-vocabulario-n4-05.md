# Vocabulario N4 — Módulo 05

:::info Propósito académico
Este vocabulario permite expresar:
- planes simples
- intención personal
- decisiones cotidianas

Criterio:
- palabras de alta frecuencia
- combinables con つもりです
- retorno directo en lectura y escucha
:::

---

## 1) Verbos típicos de intención

| Verbo | Lectura | Uso |
|---|---|---|
| 行く | いく | ir |
| 来る | くる | venir |
| 会う | あう | encontrarse |
| 勉強する | べんきょうする | estudiar |
| 始める | はじめる | empezar |
| 続ける | つづける | continuar |
| やめる | やめる | dejar |
| 休む | やすむ | descansar |

---

## 2) Expresiones temporales clave

| Expresión | Lectura | Uso |
|---|---|---|
| 今日 | きょう | hoy |
| 明日 | あした | mañana |
| 今週 | こんしゅう | esta semana |
| 来週 | らいしゅう | la próxima semana |
| 週末 | しゅうまつ | fin de semana |
| もう | もう | ya / no más |

---

## 3) Bloques listos para usar

:::info
- 今日は＿＿＿つもりです。  
- 来週は＿＿＿つもりです。  
- もう＿＿＿ないつもりです。  
- ＿＿＿から、＿＿＿つもりです。
:::

---

## 4) Microfrases modelo (referencia)

:::note
- 今日は家にいるつもりです。  
- 来週、日本語を勉強するつもりです。  
- 週末は友だちに会うつもりです。  
- 忙しいから、今日は出かけないつもりです。
:::

---

## 5) Errores típicos con vocabulario del módulo

:::warning
- Usar demasiadas expresiones de tiempo juntas
- Forzar 予定です sin necesidad
- Hacer frases largas sin control
:::
