# Práctica guiada N4 — Módulo 05

:::info Propósito académico
Esta práctica entrena:
- expresar intención con つもりです
- integrar tiempo + acción
- justificar planes de forma simple
- leer planes sin analizar cada forma

Se busca **reflejo**, no creatividad forzada.
:::

---

## 1) Activación rápida

:::note
Identifica si la frase expresa intención o solo descripción:
:::

1. 今日は家にいます。  
2. 今日は家にいるつもりです。

Marca mentalmente:
- 2 = intención clara

---

## 2) Completar con つもりです

:::info
Completa de forma natural:
:::

- 今日は＿＿＿つもりです。  
- 来週は＿＿＿つもりです。  
- 週末は＿＿＿つもりです。

Usa verbos del módulo.

---

## 3) Negación de intención

:::info
Completa:
:::

- 今日は＿＿＿ないつもりです。  
- もう＿＿＿ないつもりです。

Ejemplos posibles:
- 出かけない
- 食べない
- 飲まない

---

## 4) Intención + razón

:::info
Completa con una razón simple:
:::

- 忙しいから、＿＿＿つもりです。  
- 雨が降りそうなので、＿＿＿つもりです。

---

## 5) Microlectura (planes simples)

:::warning Regla
Lee una vez completa.
:::

> 今週は忙しいので、あまり出かけないつもりです。  
> 週末は時間があると思うので、友だちに会うつもりです。  

Preguntas (mental):
- ¿Qué planea esta semana?
- ¿Qué planea el fin de semana?

---

## 6) Producción guiada libre

:::info Tarea
Escribe 6–8 líneas simples sobre:
“Planes para hoy y la próxima semana”.

Requisitos:
- 3 frases con つもりです
- 1 frase negativa (〜ないつもりです)
- 1 razón con から o ので
:::

---

## 7) Autoevaluación inmediata

:::info Marca solo si es cierto
- ⬜ Usé つもりです sin traducir “voy a…”
- ⬜ Integré tiempo + acción naturalmente
- ⬜ Pude justificar un plan con razón breve
- ⬜ Leí el microtexto sin traducir todo
:::
