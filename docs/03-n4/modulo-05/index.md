# Módulo 05 – Acciones, intención y planes (N4)

:::info Propósito académico
Este módulo introduce y consolida un cambio fundamental en el uso del japonés:

pasar de **describir lo que haces**
a **expresar lo que vas a hacer, planeas hacer o decides hacer**.

Aquí se entrena:
- intención inmediata
- planes simples
- decisiones prácticas
- explicación breve de acciones futuras

Objetivo: que el japonés empiece a reflejar **voluntad y proyección**, no solo hechos.
:::

---

## Rol del módulo dentro del nivel N4

:::note Visión docente
En N5 predominan:
- hábitos
- rutinas
- descripciones estáticas

En N4 aparece con fuerza:
- intención
- decisión
- acción futura cercana

Este módulo prepara el terreno para:
- planes más complejos (N3)
- intención + razón + condición
:::

---

## Ejes lingüísticos del Módulo 05

:::info Contenidos integrados
:::

### 1) Gramática aplicada
- 〜つもりです (intención / plan)
- 〜予定です (plan más “objetivo” – reconocimiento)
- integración con tiempo futuro cercano
- conexión con razones simples (から／ので)

### 2) Vocabulario funcional
- verbos de acción futura (行く・会う・勉強する・始める)
- expresiones temporales (今週・来週・明日)
- decisiones cotidianas

### 3) Kanji
- 予・定・今・週・来・始 など
- reconocimiento prioritario

### 4) Lectura / microlectura
- textos cortos de “plan + intención”
- comprensión sin traducción palabra por palabra

## Cómo estudiar este módulo (regla premium)

:::warning Regla no negociable
No memorices つもりです como “voy a…”.

Debes entender:
- quién decide
- cuán firme es la intención
- si es un plan personal o algo ya fijado

El criterio importa más que la traducción.
:::

---

## Criterio de dominio del módulo

Antes de cerrar este módulo deberías poder:

- [ ] Expresar intención personal con 〜つもりです
- [ ] Distinguir intención personal de plan más objetivo (予定)
- [ ] Hablar de acciones futuras cercanas
- [ ] Integrar tiempo + acción sin esfuerzo
- [ ] Leer planes simples sin analizar cada forma

---

:::note Nota docente final
Cuando puedes hablar del futuro cercano con naturalidad,
el japonés deja de ser descriptivo y empieza a ser **activo**.
:::
