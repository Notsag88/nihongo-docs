# Gramática N4 — Experiencia y frecuencia

:::info Propósito académico
Este archivo introduce y fija:
- 〜たことがある (experiencia)
- su contraste con pasado puntual
- expresiones básicas de frecuencia

Objetivo: expresar vivencias de forma natural y controlada.
:::

---

## 1) Experiencia: 〜たことがある

:::info Función
Expresar que **alguna vez en la vida** se ha vivido algo.
El momento exacto no importa.
:::

Estructura:
- Vた + ことがある

Ejemplos:
- 日本に行ったことがあります。  
- 寿司を食べたことがあります。  
- 漫画を読んだことがあります。

:::note Matiz docente
No es “ayer”, ni “la semana pasada”.
Es **experiencia acumulada**.
:::

---

## 2) Negación de experiencia

:::info
Para decir “nunca he…”
:::

- Vたことがありません

Ejemplos:
- 日本に行ったことがありません。  
- お酒を飲んだことがありません。

---

## 3) Contraste clave: experiencia vs pasado puntual

:::info Diferencia esencial
:::

- 経験 → 〜たことがある  
- Hecho puntual → Vた

Ejemplos:
- 日本に行ったことがあります。 (experiencia)
- 去年、日本に行きました。 (evento concreto)

:::warning Error frecuente
Usar 〜たことがある cuando el tiempo es específico.
Eso rompe la lógica japonesa.
:::

---

## 4) Frecuencia básica (refuerzo N4)

:::info
Expresiones comunes:
:::

- よく (a menudo)
- ときどき (a veces)
- あまり〜ない (no muy seguido)
- ぜんぜん〜ない (nunca, en presente)

Ejemplos:
- 映画をよく見ます。  
- 外食はあまりしません。

---

## 5) Experiencia + frecuencia (natural)

:::note
Ambas pueden convivir:
:::

- 日本料理を食べたことがあります。  
- 今もときどき食べます。

---

## 6) Plantillas funcionales

:::info
- ＿＿＿たことがあります。  
- ＿＿＿たことがありません。  
- ＿＿＿をよく＿＿＿ます。  
- ＿＿＿はあまり＿＿＿ません。
:::

---

## 7) Indicadores de dominio

:::info
Está integrado si:
- no preguntas “cuándo” al usar 〜たことがある
- eliges pasado puntual solo cuando el tiempo importa
- usas frecuencia sin traducir desde español
:::
