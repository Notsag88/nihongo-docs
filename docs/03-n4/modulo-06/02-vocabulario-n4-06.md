# Vocabulario N4 — Módulo 06

:::info Propósito académico
Este vocabulario permite hablar de:
- experiencias de vida
- hábitos y frecuencia
- actividades culturales y cotidianas

Criterio:
- verbos universales
- alta frecuencia
- combinables con 〜たことがある
:::

---

## 1) Verbos típicos de experiencia

| Verbo | Lectura | Uso |
|---|---|---|
| 行く | いく | viajar / ir |
| 食べる | たべる | comer |
| 見る | みる | ver |
| 読む | よむ | leer |
| する | する | hacer |
| 住む | すむ | vivir |
| 勉強する | べんきょうする | estudiar |

---

## 2) Actividades comunes

| Actividad | Uso típico |
|---|---|
| 日本料理 | comida japonesa |
| 映画 | ver películas |
| 旅行 | viajar |
| アニメ | ver anime |
| 漫画 | leer manga |
| 温泉 | ir a onsen |

---

## 3) Frecuencia básica

:::info
- よく
- ときどき
- あまり〜ない
- ぜんぜん〜ない
:::

Ejemplos:
- アニメをよく見ます。  
- 外食はあまりしません。

---

## 4) Bloques listos para usar

:::info
- ＿＿＿に行ったことがあります。  
- ＿＿＿を食べたことがあります。  
- ＿＿＿をよく＿＿＿ます。  
- ＿＿＿はあまり＿＿＿ません。
:::

---

## 5) Microfrases modelo

:::note
- 日本料理を食べたことがあります。  
- 日本に行ったことがありません。  
- 映画をよく見ます。  
- アニメはときどき見ます。
:::

---

## 6) Errores típicos

:::warning
- Mezclar experiencia con tiempo específico
- Usar demasiados adverbios de frecuencia
- Forzar vocabulario complejo innecesario
:::
