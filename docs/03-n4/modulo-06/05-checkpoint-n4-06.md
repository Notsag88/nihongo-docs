# Checkpoint N4 — Módulo 06

:::info Propósito académico
Este checkpoint verifica dominio funcional de:
- 〜たことがある
- contraste con pasado puntual
- frecuencia básica
- lectura de experiencia personal

Este módulo cierra la **gramática N4 completa**.
:::

---

## 1) Dominio conceptual

Marca solo si puedes explicarlo:

- ⬜ 〜たことがある expresa experiencia, no tiempo
- ⬜ El pasado puntual se usa solo cuando el momento importa
- ⬜ La frecuencia describe hábitos, no experiencias pasadas
- ⬜ No mezclo experiencia y evento concreto

---

## 2) Reconocimiento rápido

Identifica:

1. 日本に行ったことがあります。 → ⬜ experiencia  
2. 去年、日本に行きました。 → ⬜ evento puntual  

---

## 3) Producción controlada

Escribe:
- ＿＿＿たことがあります。  
- ＿＿＿たことがありません。

Marca:
- ⬜ Natural
- ⬜ Sin tiempo específico

---

## 4) Frecuencia

Escribe una frase:
- ＿＿＿を＿＿＿ます。

Marca:
- ⬜ Uso correcto del adverbio

---

## 5) Lectura

Lee una vez:

> 日本に行ったことがありますが、まだ京都に行ったことがありません。  
> でも、日本文化はとても好きです。

Marca:
- ⬜ Entendí el sentido global
- ⬜ Reconocí experiencia vs contraste
- ⬜ No traduje palabra por palabra

---
