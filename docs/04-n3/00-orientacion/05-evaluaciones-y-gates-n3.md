# Evaluaciones y gates del JLPT N3  
## Cuándo se puede avanzar y cuándo no

:::info Propósito académico
Este documento define el **sistema de evaluación y gates obligatorios**
para el nivel JLPT N3.

No describe exámenes concretos.
Define **condiciones de avance**, **criterios de validación**
y **momentos donde detenerse**.

En N3, avanzar sin gates
equivale a **acumular comprensión incompleta**.
:::

---

## 1. Qué es un “gate” en este programa

:::note Definición operativa
Un **gate** es un punto de control obligatorio
que valida si una competencia está realmente consolidada
antes de avanzar.
:::

Un gate:
- no se “aprueba por intuición”
- no se supera por cantidad de contenido visto
- no se ignora sin consecuencias

---

## 2. Por qué el N3 necesita gates explícitos

En N4, muchos errores eran visibles.
En N3, los errores suelen ser:
- silenciosos
- parciales
- compensados por contexto

:::warning Riesgo principal
El estudiante cree que entiende,
pero falla en:
- inferencia
- matiz
- conclusión
- coherencia discursiva
:::

Los gates existen para **hacer visibles esos fallos**.

---

## 3. Tipos de evaluación en N3

El N3 se valida mediante **cinco tipos de evaluación**, cada uno con su gate.

---

### 3.1 Gate de gramática aplicada

Valida que el estudiante:
- reconoce estructuras en texto real
- elige estructuras según intención
- no confunde formas cercanas
- puede explicar el criterio de uso

Indicadores mínimos:
- identifica la estructura en lectura no entrenada
- justifica por qué se usa y no otra
- no responde “porque es así”

:::warning
Si la gramática solo se reconoce en ejercicios aislados,
el gate **no está superado**.
:::

---

### 3.2 Gate de lectura (comprensión profunda)

Valida que el estudiante:
- identifica idea principal
- distingue ejemplo de conclusión
- sigue el hilo lógico del texto
- infiere información implícita

Indicadores mínimos:
- resume el texto en japonés simple
- responde preguntas de intención
- no depende de traducción literal

:::note
Leer sin captar conclusión es **no pasar el gate**.
:::

---

### 3.3 Gate de escucha (intención y subtexto)

Valida que el estudiante:
- sigue audios sin pausar constantemente
- identifica intención del hablante
- reconoce finales implícitos
- tolera reformulaciones

Indicadores mínimos:
- explica qué quiso decir el hablante
- detecta cambio de postura
- no necesita el script para “entender todo”

:::warning
Entender palabras sin entender intención **no es aprobar**.
:::

---

### 3.4 Gate de producción controlada

Valida que el estudiante:
- produce frases propias
- mantiene coherencia
- usa registro neutro apropiado
- evita traducción literal

Indicadores mínimos:
- explica por qué eligió cada estructura
- corrige sus propios errores al revisarse
- no copia modelos sin adaptación

---

### 3.5 Gate de estabilidad (el más importante)

:::info Gate clave del N3
Este gate valida que el nivel es **estable en el tiempo**.
:::

Valida que:
- los mismos errores no reaparecen
- la bitácora muestra aprendizaje real
- la comprensión no se degrada con el paso de días

Indicadores mínimos:
- errores disminuyen o cambian de tipo
- lo estudiado se recuerda tras una pausa
- la confianza aumenta sin pérdida de precisión

---

## 4. Cuándo NO se puede avanzar (reglas claras)

:::warning Avance bloqueado si:
- el mismo error aparece 3 veces
- la bitácora no se revisa semanalmente
- la lectura se entiende “más o menos”
- la escucha depende del script
- la producción genera inseguridad constante
:::

En estos casos:
→ **reforzar, no avanzar**.

---

## 5. Relación entre módulos y gates

:::info Regla estructural
Cada módulo N3 **solo se considera cerrado**
cuando todos sus gates están superados.
:::

Cerrar un módulo implica:
- gate de gramática ✔️
- gate de lectura ✔️
- gate de escucha ✔️
- gate de producción ✔️
- gate de estabilidad ✔️

Fallar uno solo:
→ el módulo sigue abierto.

---

## 6. Evaluaciones formales vs gates

:::note Diferencia clave
- Las **evaluaciones** miden rendimiento puntual
- Los **gates** validan dominio sostenido
:::

Aprobar una evaluación **no habilita automáticamente** el avance
si los gates no están superados.

---

## 7. Uso de los gates para ajustar el ritmo

:::info Uso estratégico
Los gates sirven para:
- detectar sobrecarga
- decidir pausas
- ajustar ritmo
- redefinir prioridades
:::

No son castigo.
Son **herramienta de protección del aprendizaje**.

---

## 8. Gate global del nivel N3

Antes de cerrar el nivel N3, el estudiante debe poder:

- leer textos N3 sin traducción constante
- entender audios sin apoyo visual inicial
- producir textos cortos coherentes
- justificar decisiones gramaticales
- mostrar bitácora con errores trabajados

:::warning
Si alguno de estos puntos falla,
el N3 **no se considera cerrado**.
:::

---

## 9. Relación con el JLPT oficial

:::info Posicionamiento real
Un estudiante que supera los gates N3:
- está preparado para el JLPT N3
- puede adaptarse a textos nuevos
- no depende de trucos de examen
:::

La aprobación del examen es consecuencia,
no objetivo principal.

---

:::note Cierre docente
El N3 no se supera avanzando.
Se supera **consolidando**.

Los gates son la garantía
de que el progreso es real.
:::
