# Orientación del JLPT N3  
## Cómo estudiar un nivel que no admite improvisación

:::info Propósito académico
Este bloque de **Orientación N3** define **cómo debe estudiarse este nivel** dentro del programa.

No enseña japonés.
Define:
- método
- ritmo
- herramientas
- control de avance
- criterios de validación

En N3, **cómo se estudia es tan importante como qué se estudia**.
:::

---

## Por qué existe este bloque

Muchos estudiantes fracasan en N3 **no por falta de capacidad**, sino por:

- estudiar sin método
- avanzar sin consolidar
- acumular contenido sin control
- confundir exposición con dominio

:::warning Advertencia
El N3 es el nivel donde estudiar “a ojo” deja de funcionar.
:::

Este bloque existe para:
- evitar avance superficial
- ordenar la carga cognitiva
- establecer límites claros
- prevenir frustración innecesaria

---

## Qué cambia respecto a N4

En N4:
- el contenido guía
- los ejemplos sostienen
- el sentido suele estar explícito

En N3:
- el contenido exige reconstrucción
- los ejemplos no explican todo
- la inferencia es obligatoria

:::note
Por eso, el N3 **no se puede cursar sin orientación previa**.
:::

---

## Qué incluye la Orientación N3

Este bloque se compone de los siguientes documentos:

1. **Introducción al N3**  
   Cambio de nivel, exigencia real y advertencias clave.

2. **Método de estudio N3**  
   Cómo leer, escuchar y trabajar gramática densa.

3. **Ritmo y carga**  
   Cuánto estudiar, cuándo parar, cómo evitar saturación.

4. **Herramientas**  
   Uso estratégico de cuaderno, Anki, bitácoras y marcadores.

5. **Bitácora de errores**  
   Cómo registrar y explotar el error como herramienta central.

6. **Evaluaciones y gates**  
   Cómo se valida el avance y cuándo **no** se puede seguir.

## Criterio de finalización del bloque

El bloque de orientación se considera **completado** cuando el estudiante:

- entiende qué se espera en N3
- acepta el aumento de exigencia
- adopta un método activo
- se compromete a usar bitácoras y gates

:::info
Sin este compromiso, avanzar a N3 es contraproducente.
:::

---

:::note Cierre docente

un proceso sólido o una experiencia frustrante.

Aquí empieza el N3 de verdad.
:::
