# Práctica guiada de conectores lógicos N3  
## Aplicar criterio antes que forma

:::info Propósito académico
Este documento propone **práctica guiada y controlada** para consolidar
el eje de conectores lógicos del N3.

El objetivo no es producir libremente,
sino **entrenar la identificación, interpretación y elección consciente**
de relaciones lógicas.
:::

---

## 1. Regla de trabajo para esta práctica

:::warning Regla no negociable
No intentes “sonar natural”.
Intentá **ser lógico y consistente**.
:::

En esta etapa:
- se prioriza el análisis
- se justifica cada elección
- se corrige el criterio antes que la forma

---

## 2. Ejercicio A — Identificación de relación lógica

Lee cada par de ideas y respondé:
**¿Qué relación lógica se construye?**

No elijas estructura todavía.

### Ejemplo 1
- La tienda estaba llena.
- Aun así, decidió entrar.

Relación esperada:
- concesión

### Ejemplo 2
- Estudió con constancia.
- Por eso aprobó el examen.

Relación esperada:
- causa

### Ejemplo 3
- Es cómodo.
- No es muy resistente.

Relación esperada:
- contraste

:::note
Si dudás entre dos relaciones,
anotalo en la bitácora.
:::

---

## 3. Ejercicio B — Interpretación en contexto

Leé cada fragmento y respondé:
**¿Qué intención tiene el hablante?**

### Fragmento 1
El trabajo es interesante.  
Sin embargo, requiere mucho tiempo.

Pregunta:
- ¿El hablante recomienda o expresa reserva?

### Fragmento 2
Aunque es caro, se vende bien.

Pregunta:
- ¿El precio impide el resultado?

:::info
No respondas con traducción literal.
Respondé con **criterio discursivo**.
:::

---

## 4. Ejercicio C — Elección guiada de conector

Para cada caso:
1. Definí la relación lógica
2. Elegí el conector adecuado
3. Justificá la elección con una frase

### Caso 1
- El clima era malo.
- El evento se realizó.

Relación:
- concesión

Justificación esperada:
- El resultado ocurre pese a una expectativa contraria.

### Caso 2
- No tenía experiencia.
- Aun así aceptó el trabajo.

Relación:
- contraste o concesión (según interpretación)

:::note
Si dos interpretaciones son posibles,
explicá por qué elegís una.
:::

---

## 5. Ejercicio D — Corrección de interpretación

Leé la interpretación y evaluá si es correcta.

### Frase
Es práctico, pero no siempre funciona.

Interpretación dada:
- “No funciona en absoluto.”

Evaluación:
- Incorrecta

Corrección:
- La frase introduce una restricción, no una negación total.

---

## 6. Ejercicio E — Detección de error típico

Identificá el error conceptual.

### Frase
Como es barato, no lo compró.

Pregunta:
- ¿La relación lógica es coherente?

Análisis esperado:
- La relación causa no es lógica en este contexto.
- Puede tratarse de contraste o concesión mal construida.

---

## 7. Autoevaluación guiada

Respondé con honestidad:

- ¿Puedo identificar la relación sin ver la forma?
- ¿Puedo justificar por qué una interpretación es incorrecta?
- ¿Detecto restricciones implícitas?
- ¿Evito traducir automáticamente?

Si alguna respuesta es negativa,
revisá los archivos anteriores antes de avanzar.

---

## 8. Registro en la bitácora

:::info Registro obligatorio
Anotá al menos:
:::

- una confusión detectada
- una relación lógica mal interpretada
- una corrección conceptual clara

Esto consolida el aprendizaje
más que repetir ejercicios.
