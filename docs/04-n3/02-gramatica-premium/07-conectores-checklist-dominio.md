# Checklist de dominio — Conectores lógicos N3  
## Verificación de comprensión real del eje

:::info Propósito académico
Este checklist verifica si el **Eje 1 – Conectores lógicos** está
**realmente consolidado**.

No mide memoria.
Mide:
- criterio
- interpretación
- control del discurso

Si el eje no pasa este checklist,
**no se recomienda avanzar** al siguiente.
:::

---

## 1. Dominio conceptual (obligatorio)

Marcá solo si podés justificarlo sin ayuda.

- [ ] Distingo causa, contraste y concesión sin traducir
- [ ] Explico con mis palabras qué relación construye una frase
- [ ] Identifico restricciones implícitas
- [ ] Reconozco reformulaciones dentro de un texto
- [ ] Entiendo cómo los conectores organizan el discurso

:::note
Si dudás en más de un punto,
el eje aún no está cerrado.
:::

---

## 2. Comprensión en lectura (validación mínima)

Sin diccionario y sin traducir palabra por palabra, puedo:

- [ ] Seguir el hilo lógico de un texto N3
- [ ] Anticipar la dirección del discurso tras un conector
- [ ] Detectar cuándo una afirmación se limita o ajusta
- [ ] Diferenciar idea principal de aclaración
- [ ] Detectar interpretaciones incorrectas en opciones múltiples

---

## 3. Comprensión en escucha (criterio real)

En audio N3, puedo:

- [ ] Identificar relaciones lógicas aunque no estén enfatizadas
- [ ] Reconocer concesiones por contexto y tono
- [ ] No interpretar restricciones como absolutos
- [ ] Mantener coherencia global del mensaje
- [ ] Justificar por qué una respuesta es correcta

---

## 4. Producción controlada (sin improvisación)

En producción guiada, puedo:

- [ ] Definir primero la relación lógica
- [ ] Elegir un conector acorde a esa relación
- [ ] Explicar por qué esa elección es adecuada
- [ ] Corregir frases con conectores mal usados
- [ ] Evitar sobrecargar una frase con conectores

:::warning
Si producís sin justificar,
no se considera dominio.
:::

---

## 5. Detección y corrección de errores

Puedo identificar y corregir:

- [ ] Traducciones automáticas incorrectas
- [ ] Confusión entre causa y concesión
- [ ] Contrastes mal interpretados
- [ ] Restricciones ignoradas
- [ ] Reformulaciones no detectadas

Cada error identificado debe tener:
- causa
- corrección
- ejemplo revisado

---

## 6. Uso de la bitácora (obligatorio)

- [ ] Registré al menos dos errores reales de este eje
- [ ] Anoté la corrección conceptual
- [ ] Puedo explicar qué me confundió y por qué
- [ ] Veo reducción de errores similares en textos nuevos

Sin bitácora, el eje no se considera cerrado.

---

## 7. Gate de avance (decisión consciente)

:::warning Gate N3
Avanzo al siguiente eje solo si:
:::

- [ ] Marco al menos el 90% de este checklist
- [ ] Puedo justificar cada ítem marcado
- [ ] No dependo de traducción para entender conectores
- [ ] No arrastro confusiones conceptuales

Si no se cumple:
→ reforzar los archivos del eje y repetir práctica.

---

:::note Cierre docente
Dominar conectores lógicos es dominar
la **arquitectura del discurso japonés**.

Este eje define si el N3
se vuelve comprensible
o una acumulación de frases sueltas.
:::
