# Checklist de dominio — Modalización N3  
## Validación real del Eje 2

:::info Propósito académico
Este checklist verifica si la **modalización N3**
está realmente consolidada.

No mide memoria.
Mide **criterio, interpretación y control discursivo**.
:::

---

## 1. Dominio conceptual

- [ ] Distingo probabilidad de suposición
- [ ] Identifico opinión mitigada
- [ ] Reconozco reserva y distancia del hablante
- [ ] Explico el grado de compromiso en una frase
- [ ] No leo conjeturas como hechos

---

## 2. Lectura N3

- [ ] Interpreto correctamente posturas del hablante
- [ ] No exagero conclusiones
- [ ] Distingo hechos de evaluaciones
- [ ] Justifico interpretaciones con contexto
- [ ] Detecto reservas implícitas

---

## 3. Escucha N3

- [ ] Reconozco modalización por tono y contexto
- [ ] No interpreto afirmaciones débiles como fuertes
- [ ] Mantengo coherencia global del mensaje
- [ ] Justifico respuestas en audio
- [ ] Identifico compromiso al final del enunciado

---

## 4. Producción controlada

- [ ] Elijo conscientemente cuánto afirmar
- [ ] Evito compromiso excesivo
- [ ] Mantengo coherencia en una frase
- [ ] Justifico elecciones gramaticales
- [ ] No uso modalización como muletilla

---

## 5. Bitácora de errores

- [ ] Registré errores reales de este eje
- [ ] Anoté correcciones conceptuales
- [ ] Veo reducción de errores similares
- [ ] Puedo explicar qué cambió en mi lectura

---

## 6. Gate de avance

:::warning Gate N3
Avanzo solo si:
:::

- [ ] Cumplo al menos el 90% del checklist
- [ ] Justifico cada ítem marcado
- [ ] No arrastro confusiones de compromiso
- [ ] Interpreto textos N3 con matiz correcto

Si no se cumple:
→ reforzar antes de avanzar.

---

:::note Cierre docente
La modalización define el **tono adulto del japonés N3**.

Dominar este eje es entender
no solo qué se dice,
sino **con cuánta responsabilidad se dice**.
:::
