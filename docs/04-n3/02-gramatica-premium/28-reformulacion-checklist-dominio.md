# Checklist de dominio — Reformulación y control del discurso (N3)  
## Validación final del Eje 4

:::info Propósito académico
Este checklist valida el dominio real del **Eje 4 — Reformulación, precisión y guía del discurso**.

Este eje no evalúa conocimiento gramatical aislado,
sino la capacidad de:
- jerarquizar información
- ajustar el alcance del mensaje
- guiar la interpretación del interlocutor

Sin este dominio, el japonés N3 queda incompleto.
:::

---

## 1. Dominio conceptual (obligatorio)

Marcá solo si podés explicarlo con tus palabras:

- [ ] Distingo claramente idea núcleo de aclaración
- [ ] Entiendo que reformular no es repetir
- [ ] Reconozco cuándo una frase ajusta el alcance de otra
- [ ] Identifico precisión y delimitación como funciones discursivas
- [ ] Comprendo que la segunda formulación suele ser la más importante

---

## 2. Lectura N3 (jerarquía y sentido)

En textos N3, puedo:

- [ ] Identificar la idea principal real del texto
- [ ] No confundir ejemplos con conclusiones
- [ ] Detectar aclaraciones que cambian el alcance
- [ ] Reinterpretar el texto al encontrar una reformulación
- [ ] Explicar por qué una lectura literal es incorrecta

---

## 3. Escucha N3 (control del discurso)

En audio N3, puedo:

- [ ] Seguir reformulaciones rápidas sin perder el hilo
- [ ] Detectar aclaraciones dichas con tono bajo
- [ ] Reconocer cuándo el hablante guía la interpretación
- [ ] No tratar reformulaciones como información nueva
- [ ] Justificar respuestas por jerarquía del mensaje

---

## 4. Producción controlada (precisión responsable)

En producción guiada, puedo:

- [ ] Reformular solo cuando es necesario
- [ ] Precisar información sin rigidizar el discurso
- [ ] Delimitar alcance para evitar ambigüedad
- [ ] Guiar al interlocutor sin redundancia
- [ ] Mantener coherencia entre núcleo y aclaración

:::warning
Si reformulás sin poder justificar el porqué,
no se considera dominio.
:::

---

## 5. Errores y bitácora (obligatorio)

- [ ] Registré errores de jerarquía real
- [ ] Identifiqué cuándo confundí ejemplo con núcleo
- [ ] Anoté correcciones conceptuales
- [ ] Puedo explicar qué parte del texto era aclaratoria
- [ ] Veo mejora en lecturas nuevas

---

## 6. Gate de avance (decisión académica)

:::warning Gate N3
Avanzo solo si:
:::

- [ ] Cumplo al menos el 90% del checklist
- [ ] Justifico cada ítem marcado
- [ ] No interpreto textos N3 de forma plana
- [ ] Leo con jerarquía y precisión natural

Si no se cumple:
→ reforzar archivos 22 a 27 antes de avanzar.

---

:::note Cierre docente
El Eje 4 define si el estudiante
puede manejar discurso japonés complejo.

Reformular bien es pensar bien en japonés.
:::
