# Gramática Premium del JLPT N3  
## Síntesis estructural y dominio funcional

:::info Propósito académico
Este documento sintetiza **toda la Gramática Premium N3**
como un **sistema coherente**, no como una lista de temas.

Su objetivo es consolidar:
- visión global
- relaciones entre ejes
- criterio de uso real

Si este documento no se entiende,
la gramática N3 aún no está integrada.
:::

---

## 1. Qué significa dominar la gramática N3

:::note Definición docente
Dominar la gramática N3 no es:
- saber muchas formas
- reconocer estructuras sueltas

Es:
- interpretar intención
- evaluar alcance
- controlar compromiso
- ajustar discurso
:::

---

## 2. Los cuatro ejes como sistema

La gramática premium N3 se organiza en **cuatro ejes funcionales**:

### Eje 1 — Conectores lógicos  
Organiza relaciones entre ideas.

### Eje 2 — Modalización  
Define cuánto se compromete el hablante.

### Eje 3 — Restricción y excepción  
Delimita el alcance de las afirmaciones.

### Eje 4 — Reformulación y precisión  
Ajusta, aclara y guía la interpretación.

Ningún eje funciona aislado.

---

## 3. Cómo interactúan los ejes en textos reales

:::info Visión integradora
En textos N3 reales:
:::

- un conector organiza
- una modalización suaviza
- una restricción delimita
- una reformulación aclara

El sentido final surge
de la interacción de todos.

---

## 4. El error del enfoque fragmentado

:::warning Error típico
Estudiar cada punto gramatical
como si fuera independiente.
:::

Esto produce:
- comprensión superficial
- errores en lectura larga
- producción poco natural

La gramática N3 es **arquitectura**, no inventario.

---

## 5. Gramática y habilidades integradas

:::info Relación directa
La gramática premium N3:
:::

- define la comprensión lectora
- condiciona la escucha
- estructura la producción
- permite inferencia correcta

No es una sección más:
es el **motor del nivel**.

---

## 6. Indicadores de integración real

Podés decir que integraste la gramática N3 si:

- entendés textos sin traducir palabra por palabra
- detectás intención y límites
- evitás conclusiones extremas
- justificás interpretaciones
- producís con precisión y control

---

## 7. Gramática N3 como puente al N2

:::note Proyección académica
El N3 es el punto de inflexión.
:::

Aquí se define si el japonés:
- se vuelve sistema claro
- o se vuelve carga confusa

Una gramática N3 sólida
hace que el N2 sea exigente pero abordable.

---

:::note Cierre docente
La gramática premium N3
no se memoriza.

Se **integra**, se **razona**
y se **usa con criterio**.

Eso es japonés real.
:::
