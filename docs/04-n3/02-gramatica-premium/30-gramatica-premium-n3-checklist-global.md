# Checklist global de dominio — Gramática Premium N3  
## Gate académico final del nivel

:::info Propósito académico
Este checklist valida si la **Gramática Premium N3**
está realmente dominada a nivel funcional.

No es una autoevaluación superficial.
Es un **gate académico** antes de avanzar a:
- Lecturas Premium N3
- Escucha Premium N3
- Producción avanzada
- Preparación seria al N2
:::

---

## 1. Comprensión global

Puedo afirmar con honestidad que:

- [ ] Entiendo textos N3 sin traducir palabra por palabra
- [ ] Identifico intención del hablante
- [ ] Distingo hechos, opiniones y conjeturas
- [ ] Detecto límites y excepciones
- [ ] Reinterpreto el texto cuando aparece una aclaración

---

## 2. Lectura JLPT N3

En lectura, puedo:

- [ ] Elegir opciones moderadas frente a absolutas
- [ ] Justificar respuestas con estructura del texto
- [ ] No perderme en ejemplos secundarios
- [ ] Explicar por qué una opción exagera el alcance
- [ ] Mantener coherencia global del texto

---

## 3. Escucha JLPT N3

En escucha, puedo:

- [ ] Seguir reformulaciones rápidas
- [ ] Captar restricciones tardías
- [ ] Detectar reservas por tono y contexto
- [ ] No interpretar conjeturas como hechos
- [ ] Justificar respuestas con intención y jerarquía

---

## 4. Producción N3 (control real)

En producción guiada, puedo:

- [ ] Elegir cuánto comprometerme al hablar
- [ ] Delimitar afirmaciones correctamente
- [ ] Reformular para aclarar sin redundancia
- [ ] Evitar afirmaciones innecesariamente fuertes
- [ ] Mantener coherencia discursiva

---

## 5. Errores y bitácora

- [ ] Mantengo una bitácora activa de errores
- [ ] Registro errores conceptuales, no solo de forma
- [ ] Veo patrones de error y su corrección
- [ ] Puedo explicar qué cambió en mi comprensión
- [ ] Uso la bitácora como herramienta real de mejora

---

## 6. Gate final N3

:::warning Gate académico
Considero la Gramática Premium N3 cerrada solo si:
:::

- [ ] Cumplo al menos el 90% del checklist
- [ ] Justifico cada ítem marcado
- [ ] No dependo de traducción literal
- [ ] Leo y escucho con criterio y control
- [ ] Produzco japonés preciso y natural

Si no se cumple:
→ volver a los ejes correspondientes.

---

:::note Cierre docente final
El N3 no se aprueba por acumulación.
Se supera por **criterio lingüístico**.

Si llegaste hasta acá con dominio real,
estás preparado para el japonés avanzado.
:::
