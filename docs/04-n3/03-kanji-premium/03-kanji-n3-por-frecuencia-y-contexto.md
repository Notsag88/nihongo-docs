# Kanji N3 por frecuencia y contexto  
## Priorizar lo que realmente se lee

:::info Propósito académico
Este documento clasifica los kanji N3
según **frecuencia real de aparición y contexto**,
para optimizar el estudio y evitar dispersión.
:::

---

## Categoría 1 — Alta frecuencia transversal

Aparecen **en casi todos los tipos de textos N3**.

| Kanji | Función | Ejemplos |
|------|--------|---------|
| 状 | estado | 状況 |
| 性 | propiedad | 可能性 |
| 的 | cualidad | 基本的 |
| 対 | relación | 対象 |
| 必 | necesidad | 必要 |

:::note
Estos kanji deben reconocerse **automáticamente**.
:::

---

## Categoría 2 — Frecuencia media contextual

Aparecen con frecuencia,
pero ligados a ciertos temas.

| Kanji | Contexto típico | Ejemplos |
|------|----------------|---------|
| 制 | sistemas | 制度 |
| 変 | procesos | 変化 |
| 増 | datos / cantidad | 増加 |
| 減 | datos / cantidad | 減少 |
| 続 | procesos | 継続 |

---

## Categoría 3 — Baja frecuencia (reconocimiento pasivo)

Aparecen ocasionalmente.
No bloquean lectura global.

| Kanji | Uso | Ejemplo |
|------|-----|--------|
| 例 | ejemplo | 例外 |
| 範 | ámbito | 範囲 |
| 限 | límite | 制限 |

:::warning
No memorizar activamente.
Reconocer cuando aparezcan.
:::

---

## Error típico del estudiante

:::warning Error frecuente
Estudiar primero kanji de baja frecuencia
por parecer “difíciles”.
:::

Resultado:
- mucho esfuerzo
- poco impacto real

---

## Uso práctico de esta clasificación

:::info Método recomendado
:::

- alta frecuencia → automatizar
- media frecuencia → reforzar en lectura
- baja frecuencia → solo reconocimiento contextual

---

:::note Cierre docente
La frecuencia no es una regla rígida,
es una **herramienta de enfoque inteligente**.
:::
