# Checklist de Dominio – Kanji Premium N3  
## Validación académica antes de avanzar

:::info Propósito académico
Este documento no es un resumen.
Es un **instrumento de validación real** del dominio de kanji en nivel N3.

Su función es determinar si:
- el estudio fue funcional
- la lectura mejoró de forma objetiva
- el kanji dejó de ser un obstáculo

Si este checklist no se cumple,
**no se recomienda avanzar**.
:::

---

## 1. Dominio de reconocimiento (criterio central)

Marcá como **logrado** solo si se cumple **sin esfuerzo consciente**:

- [ ] Reconozco kanji N3 frecuentes sin detener la lectura  
- [ ] Identifico rápidamente compuestos abstractos (〜性, 〜化, 〜的, etc.)  
- [ ] No necesito traducir carácter por carácter  
- [ ] Puedo inferir significado general por contexto  
- [ ] Distingo función del kanji dentro del compuesto  

:::warning
Si todavía “leo lento pero entiendo”,
el reconocimiento **no está automatizado**.
:::

---

## 2. Dominio de patrones y combinaciones

- [ ] Reconozco patrones comunes (名詞＋的, 〜化, 〜性, 原因→結果)  
- [ ] Leo compuestos largos sin fragmentarlos mentalmente  
- [ ] Identifico verbo núcleo en frases complejas  
- [ ] Anticipo significado por estructura del compuesto  

:::note
Este punto es clave para lectura fluida en N3.
:::

---

## 3. Sustantivos abstractos y discurso

- [ ] Entiendo conceptos abstractos sin buscar traducción exacta  
- [ ] Distingo causa, efecto, relación y sistema en textos  
- [ ] Reconozco evaluación y postura del autor  
- [ ] No me bloqueo ante vocabulario no concreto  

---

## 4. Adjetivos, estados y evaluación

- [ ] Interpreto estados como relativos, no absolutos  
- [ ] Identifico matices de evaluación (adecuado, suficiente, limitado)  
- [ ] Relaciono estado con proceso previo  
- [ ] Capto conclusiones correctamente  

---

## 5. Reconocimiento vs producción (criterio avanzado)

- [ ] No intento escribir todos los kanji  
- [ ] Distingo claramente kanji de reconocimiento y de producción  
- [ ] Uso escritura solo cuando refuerza distinciones clave  
- [ ] No siento culpa por “no saber escribir” un kanji leído  

:::warning Regla de madurez
En N3, escribir menos y leer mejor
es **progreso**, no carencia.
:::

---

## 6. Uso correcto de Anki / SRS

- [ ] Anki refuerza lectura, no la reemplaza  
- [ ] Uso tarjetas mayormente de reconocimiento  
- [ ] Estudio kanji siempre con contexto real  
- [ ] No sobrecargo el sistema con listas masivas  

---

## 7. Indicadores globales de progreso real

Respondé con honestidad:

- [ ] Leo textos N3 con menos fricción que antes  
- [ ] Uso menos el diccionario  
- [ ] Entiendo el sentido general aunque falten detalles  
- [ ] La lectura ya no me resulta intimidante  
- [ ] Puedo sostener lectura continua sin agotarme  

Si la mayoría es **sí**,
el módulo está correctamente trabajado.

---

## Decisión académica

:::info Gate de avance
Podés avanzar a **Lecturas Premium N3** solo si:
:::

- el reconocimiento es automático
- los patrones están internalizados
- el kanji ya no bloquea comprensión

Si no:
→ volver a lectura guiada  
→ reforzar patrones  
→ ajustar Anki  
→ **no avanzar por presión de calendario**

---

:::note Cierre docente final
Kanji Premium N3 no busca que “sepas muchos kanji”.
Busca que **el japonés escrito deje de ser una barrera**.

Si llegaste hasta acá con control y claridad,
estás listo para enfrentar textos N3 reales.
:::
