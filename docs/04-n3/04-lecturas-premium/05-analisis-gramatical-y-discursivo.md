# Análisis gramatical y discursivo en lecturas N3  
## Ver la estructura que sostiene el texto

:::info Propósito académico
Este archivo enseña **cómo analizar un texto N3 después de leerlo**,
sin romper la continuidad lectora.

El análisis sirve para:
- confirmar comprensión
- reforzar patrones
- mejorar futuras lecturas
:::

---

## Qué se analiza en N3 (y qué no)

:::note Criterio docente
En N3 no se analiza:
:::

- cada palabra
- cada forma gramatical

Se analiza:
- **la función** de la gramática
- **la estructura** del discurso

---

## Elementos gramaticales clave a observar

### 1. Conectores lógicos

- そのため (causa → resultado)
- 一方で (contraste)
- しかし (oposición)

Función:
→ organizar el flujo del texto

---

### 2. Expresiones de evaluación

- 重要だと考えられている
- 面倒だと感じる
- 求められる

Estas expresiones:
→ señalan postura o juicio

---

### 3. Uso del pasivo y lo impersonal

:::info Observación
El N3 usa frecuentemente:
:::

- pasivo
- expresiones sin sujeto claro

Esto desplaza el foco
del actor al sistema.

---

## Kanji como soporte del discurso

:::note Integración
Kanji abstracto como:
:::

- 制度
- 効率
- 結果
- 利益

Permite identificar:
- tema
- proceso
- conclusión

---

## Error típico en el análisis

:::warning Error frecuente
Analizar antes de entender el texto.
:::

El orden correcto es:
1. lectura global
2. comprensión estructural
3. análisis selectivo

---

## Cómo usar este análisis en la práctica

:::info Método recomendado
Después de cada lectura:
:::

- subrayar conectores
- marcar evaluación
- identificar conclusión
- anotar 1–2 patrones reutilizables

Nada más.

---

:::note Cierre docente
El análisis gramatical N3
no busca desarmar el texto,
sino **hacer visible su arquitectura**.
:::
