# Análisis auditivo y discursivo en Escucha N3  
## Confirmar comprensión sin romper la escucha

:::info Propósito académico
Este archivo enseña **cómo analizar un audio N3 después de escucharlo**,
para reforzar patrones y mejorar escuchas futuras,
sin caer en traducción literal.
:::

---

## Qué se analiza (y qué no)

:::note Criterio docente
En N3 no se analiza:
:::

- cada palabra
- cada forma gramatical

Se analiza:
- **intención**
- **estructura**
- **señales discursivas**

---

## Elementos clave a identificar

### 1. Intención del hablante

- informar
- opinar
- proponer
- decidir

Reconocer la intención
aclara todo el audio.

---

### 2. Estructura del discurso

:::info Patrones frecuentes
:::

- situación → problema → solución
- causa → resultado → evaluación
- opinión → contraste → conclusión

---

### 3. Señales auditivas estructurales

| Expresión | Función |
|---------|---------|
| そのため | causa → efecto |
| しかし | contraste |
| つまり | conclusión |
| 最終的に | cierre |

Estas palabras **ordenan el audio**.

---

## Kanji y lectura como soporte indirecto

:::note Integración
La comprensión auditiva mejora cuando:
:::

- los patrones ya fueron leídos
- el vocabulario ya fue reconocido visualmente

Escucha y lectura se refuerzan mutuamente.

---

## Error típico en el análisis

:::warning Error frecuente
Analizar mientras se escucha.
:::

Primero:
1. escuchar
2. comprender globalmente

Después:
3. analizar selectivamente

---

## Método práctico post-escucha

:::info Procedimiento recomendado
:::

- anotar intención
- marcar estructura
- identificar conclusión
- registrar 1–2 expresiones clave

Nada más.

---

:::note Cierre docente
El análisis auditivo N3
no busca desarmar el audio,
sino **hacer visible su arquitectura**.
:::
