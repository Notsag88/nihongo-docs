# Vocabulario N3  
## Organización y contraste

:::info Objetivo léxico
Trabajar vocabulario que **articula ideas**,
no objetos concretos.
:::

---

## Vocabulario clave

| Palabra | Significado |
|-------|-------------|
| 方法 | método |
| 比較 | comparación |
| 違い | diferencia |
| 同じ | igual |
| 問題 | problema |
| 点 | punto / aspecto |
| 場合 | caso / situación |

---

## Uso esperado

Estas palabras aparecen:
- en explicaciones
- en argumentos
- en comparaciones

Deben reconocerse **sin pausa**.

---

## Error típico

:::warning Error frecuente
Intentar memorizar definiciones largas.
:::

El vocabulario N3 se fija **por uso repetido**.
