# Gramática aplicada N3  
## Expresar opinión con matiz

:::info Objetivo gramatical
Consolidar estructuras N3 usadas para:
- expresar opinión
- evaluar situaciones
- suavizar afirmaciones
:::

---

## Estructuras trabajadas

| Estructura | Función |
|-----------|---------|
| 〜と思う | opinión personal |
| 〜と考える | opinión reflexiva |
| 〜ように感じる | percepción subjetiva |
| 〜と言われている | evaluación general |

---

## Ejemplo integrado

> この方法は効果的だ**と思います**。  
> ただ、人によっては難しい**と感じる**かもしれません。  
> 一般的には良い方法だ**と言われています**。

---

## Diferencia clave

:::note Observación docente
:::

- 〜と思う → postura directa  
- 〜と言われている → distancia / generalización  

Elegir mal **altera el tono**.

---

## Error típico N3

:::warning Error frecuente
Afirmar demasiado fuerte.
:::

El N3 valora **matiz**, no certeza absoluta.
