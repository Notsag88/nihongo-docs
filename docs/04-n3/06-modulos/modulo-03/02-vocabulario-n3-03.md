# Vocabulario N3  
## Evaluar, juzgar y matizar

:::info Objetivo léxico
Reconocer vocabulario usado para:
- evaluar
- opinar
- describir estados abstractos
:::

---

## Vocabulario clave

| Palabra | Significado |
|-------|-------------|
| 意見 | opinión |
| 考え | pensamiento |
| 感じ | sensación |
| 効果 | efecto |
| 印象 | impresión |
| 状態 | estado |
| 判断 | juicio |

---

## Uso esperado

Este vocabulario aparece:
- en textos de opinión
- en evaluaciones
- en conversaciones reflexivas

Debe reconocerse **sin pausa**.
