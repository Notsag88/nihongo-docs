# Gramática aplicada N3  
## Explicar por qué ocurre algo

:::info Objetivo gramatical
Consolidar el uso funcional de estructuras N3
para expresar:
- causa
- razón
- consecuencia
:::

---

## Estructuras trabajadas

| Estructura | Función |
|-----------|---------|
| 〜から | causa directa |
| 〜ので | causa objetiva / suave |
| そのため | consecuencia |
| 〜結果 | resultado formal |

---

## Ejemplo integrado

> 雨が降った**ので**、試合は中止になりました。  
> **そのため**、多くの人が予定を変更しました。  
> **結果**として、会場は静かでした。

---

## Diferencia clave

:::note Observación docente
:::

- から → causa subjetiva / directa  
- ので → causa objetiva / explicativa  

Elegir mal afecta el **tono**.

---

## Error típico N3

:::warning Error frecuente
Usar todas las causas como si fueran iguales.
:::

El N3 evalúa **matiz causal**, no solo forma.
