# Vocabulario N3  
## Explicar relaciones y resultados

:::info Objetivo léxico
Reconocer vocabulario usado para:
- explicar razones
- describir consecuencias
- justificar resultados
:::

---

## Vocabulario clave

| Palabra | Significado |
|-------|-------------|
| 理由 | razón |
| 原因 | causa |
| 結果 | resultado |
| 影響 | influencia |
| 変化 | cambio |
| 関係 | relación |

---

## Uso esperado

Estas palabras aparecen:
- en textos explicativos
- en noticias simples
- en razonamientos cotidianos
