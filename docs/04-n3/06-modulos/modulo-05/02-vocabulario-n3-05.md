# Vocabulario N3  
## Describir evolución y transformación

:::info Objetivo léxico
Reconocer vocabulario usado para:
- procesos
- cambios graduales
- desarrollo continuo
:::

---

## Vocabulario clave

| Palabra | Significado |
|-------|-------------|
| 変化 | cambio |
| 成長 | crecimiento |
| 進歩 | progreso |
| 過程 | proceso |
| 段階 | etapa |
| 状態 | estado |

---

## Uso esperado

Este vocabulario aparece:
- en explicaciones
- en descripciones de evolución
- en textos narrativos no ficcionales
