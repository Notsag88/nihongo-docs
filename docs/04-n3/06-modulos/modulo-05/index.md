# Módulo 05  
## Comprender transformaciones y evolución en N3

:::info Propósito del módulo
Este módulo entrena una competencia clave del N3:
**seguir procesos y cambios de manera continua**.

Aquí el japonés se usa para:
- describir evolución
- explicar transformaciones
- entender desarrollo progresivo
:::

---

## Qué se entrena en este módulo

- verbos y expresiones de cambio
- secuenciación temporal
- comprensión de procesos completos
- continuidad en textos y audios dinámicos

---

## Contenidos integrados

- Gramática: cambio, desarrollo y transición
- Vocabulario: léxico de proceso
- Kanji: transformación, avance y estado
- Lectura: texto de desarrollo progresivo
- Escucha: explicación de proceso
- Práctica: descripción integrada

---

## Regla del módulo

:::warning Regla no negociable
Si no podés explicar:
:::

- qué cambia
- cómo cambia
- en qué orden

→ el módulo **no está integrado**.

---

## Cierre esperado

Al finalizar, deberías poder:
- seguir procesos sin perderte
- describir cambios con claridad
- mantener continuidad cognitiva
