# Gramática aplicada N3  
## Expresar intención y elección

:::info Objetivo gramatical
Consolidar estructuras N3 usadas para:
- expresar intención
- indicar decisión
- justificar elecciones
:::

---

## Estructuras trabajadas

| Estructura | Función |
|-----------|---------|
| 〜つもりだ | intención |
| 〜ことにする | decisión |
| 〜ようと思う | intención reflexiva |
| 〜ために | finalidad |

---

## Ejemplo integrado

> 来年、日本に留学する**つもりです**。  
> いろいろ考えた**結果**、この学校に行く**ことにしました**。  
> 日本語をもっと使える**ように**なる**ために**、毎日勉強しています。

---

## Diferencia clave

:::note Observación docente
:::

- つもりだ → plan previo  
- ことにする → decisión tomada  

Confundirlos cambia el **momento de la decisión**.

---

## Error típico N3

:::warning Error frecuente
Usar todas las decisiones como intención vaga.
:::

El N3 exige **precisión temporal y mental**.
