# Lectura guiada N3  
## Comprender proceso de elección

:::info Objetivo de la lectura
Leer un texto N3 identificando:
- opciones
- criterio de elección
- decisión final
:::

---

## Texto

仕事を続けるか、勉強を続けるか、長い間迷っていました。  
いろいろ考えた結果、勉強を優先することにしました。  
将来、日本語を仕事で使えるようになるためです。  
この選択は簡単ではありませんでしたが、後悔はありません。

---

## Guía de lectura

- ¿Qué opciones había?
- ¿Qué criterio se usa para decidir?
- ¿Cuál es la intención final?

---

## Error típico

:::warning Error frecuente
Leer solo el resultado y no el proceso.
:::

Este texto evalúa la **justificación**, no solo la decisión.
