# Gramática aplicada N3  
## Expresar límites y condiciones

:::info Objetivo gramatical
Consolidar estructuras N3 usadas para:
- expresar condición
- establecer restricciones
- marcar límites claros
:::

---

## Estructuras trabajadas

| Estructura | Función |
|-----------|---------|
| 〜場合は | en caso de |
| 〜なければならない | obligación |
| 〜てはいけない | prohibición |
| 〜しか〜ない | limitación |

---

## Ejemplo integrado

> 雨が降った**場合は**、試合は中止になります。  
> 参加者は、ルールを守ら**なければなりません**。  
> 危険な物を持ち込ん**ではいけません**。  
> この方法**しか**選べ**ません**。

---

## Observación clave

:::note Observación docente
:::

Las estructuras de restricción:
- no describen hechos
- **controlan acciones**

Elegir mal genera ambigüedad.

---

## Error típico N3

:::warning Error frecuente
Usar condición como simple tiempo.
:::

En N3, la condición **define posibilidad**, no solo momento.
