# Checkpoint académico  
## Validación del Módulo 08

:::info Criterio de validación
Este checkpoint confirma dominio de:
- restricción
- condición
- límite y excepción
:::

---

## Dominio esperado

- [ ] Entiendo reglas al leer
- [ ] Comprendo condiciones al escuchar
- [ ] Distingo obligación y prohibición
- [ ] Expreso límites con claridad
- [ ] Manejo excepciones correctamente

---

## Decisión

:::warning Gate
Si 2 o más ítems no están logrados:
:::

→ repetir el módulo  
→ no avanzar
