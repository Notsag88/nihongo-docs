# Checkpoint académico final  
## Validación del N3 integrado

:::info Criterio de validación
Este checkpoint define si el **N3 está cerrado**.
:::

---

## Dominio esperado

- [ ] Leo textos N3 tipo examen con estabilidad
- [ ] Escucho audios N3 sin pánico
- [ ] Integro gramática sin analizar
- [ ] El kanji no bloquea
- [ ] Manejo tiempo y presión

---

## Decisión final

:::warning Gate definitivo
Si 2 o más ítems no están logrados:
:::

→ reforzar módulos previos  
→ **no avanzar a N2**

---

:::note Cierre docente final
Si este módulo se supera con control,
el N3 deja de ser un objetivo
y pasa a ser una **base sólida para N2**.
:::
