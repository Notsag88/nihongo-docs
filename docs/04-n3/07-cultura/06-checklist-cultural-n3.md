# Checklist cultural N3

:::info Propósito

## Dominio esperado

- [ ] Acepto ambigüedad funcional
- [ ] Comprendo pasivo como neutralidad
- [ ] Leo reglas como sistemas
- [ ] No busco confrontación donde no existe
- [ ] Interpreto intención sin que se diga

## Decisión

→ **no avanzar a N2**
