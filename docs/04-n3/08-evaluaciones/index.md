# Evaluaciones N3  
## Verificación de dominio real del nivel

:::info Propósito académico
Las evaluaciones N3 **no miden memoria**.
Miden si el japonés:
- se comprende de forma estable
- se integra sin análisis consciente
- resiste lectura y escucha prolongada
:::

---

## Qué se evalúa realmente en N3

En este nivel se evalúa:

- comprensión global
- inferencia
- interpretación cultural
- tolerancia a la ambigüedad
- estabilidad bajo carga

No se evalúa:
- traducción palabra por palabra
- memorización aislada
- análisis gramatical explícito

---

## Tipos de evaluación en este bloque

Este bloque incluye:

- diagnósticos de integración
- evaluaciones por habilidad
- evaluación cultural
- simulacro premium tipo JLPT
- checklist final de dominio

Cada uno cumple una función distinta.

---

## Regla general de uso

:::warning Regla no negociable
No repitas una evaluación para “mejorar la nota”.
:::

Si fallás:
→ volvé al bloque correspondiente  
→ reforzá  
→ **reintentá después de días**, no de horas

---

## Objetivo final

Estas evaluaciones determinan si:
- el N3 está cerrado
- el estudiante puede avanzar a N2
- el japonés funciona como idioma real
